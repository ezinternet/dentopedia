비니어 삭제량 계산기 — “Do the Math”

목업/APT를 관통해 삭제 · 부위별 P = max(0, LT − EV) · 법랑질 보존 의사결정 보조 (Coachman 2014, Gürel 2007)

1 목표 색 변화 (shade tone) 선택

규칙 출처: Coachman 2014 — 1–2 tone 변화 → 박형 ~0.3mm 라미네이트; ≥3 tone 또는 현저한 변색 → 더 두꺼운 도재 + intrasulcular 변연으로 색 전이 은폐. “더 어두운 치아 = 더 많은 삭제”의 근거.

2 부위별 추가부피(EV) 입력 → 삭제량(P) 산출

EV = 부가형 진단 왁스업/목업이 원래 치아 표면 밖으로 더한 부피(mm). 술자가 caliper/실리콘 인덱스 단면으로 실측해 입력. LT = 목표색에 필요한 라미네이트 두께(색 선택 시 자동 채움, 수정 가능).

부위EV (mm)LT (mm)P = max(0, LT−EV)판정 / 버
수식 설명 목업면(=환자 승인 최종 윤곽)에서 보철 안쪽면까지 거리가 도재 두께 LT. 목업면에서 원래 치아까지가 추가부피 EV. 따라서 LT = EV + P → P = LT − EV. EV ≥ LT이면 추가부피만으로 도재 공간이 확보되어 삭제 0(법랑질 무손상). 음수는 0으로 클램프. (Coachman 2014 Table 1 재현)

3 무삭제(No-Prep) 금기 체크

출처: Chandode 2026 — 무삭제 금기 = 내인성 변색·부정정렬·큰 형태 수정. 무삭제·통상 모두 91–100% 생존(법랑질 본딩 시 무삭제 성공률 ≤97.4%)이나 적응증이 갈림(경쟁 아닌 상호보완).

📋 결과 요약

⚠ 비협상 원칙 — 법랑질 한정 변연. 모든 변연을 법랑질에 두십시오. 상아질 변연 노출 시 실패율 약 10배(El-Mowafy 2018, Gürel 580 비니어 12년). 계산된 P가 법랑질 두께를 초과하거나 변연이 상아질에 떨어지면 — 제약이 아니라 디자인(목업 부피·변연 위치·재료)을 바꾸십시오.
최소삭제(MPV 0.2–0.5mm)는 전통삭제(CV 0.3–1.0mm) 대비 생존 동등 이상(Ali 2023, SR 4편). 초박형 contact-lens형은 마취·임시치 불필요. 단 비교 RCT는 아직 없음.
