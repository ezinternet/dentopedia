# Part 1 — ISQ란

---

## ISQ = Implant Stability Quotient

- **공명주파수분석**(Resonance Frequency Analysis, RFA)으로 측정
- 1–100 수치, 임플란트의 미세 흔들림(micromotion)에 반비례
- 두 기기: **Osstell** (자기장 SmartPeg) vs **Osseo** (전자기 sensor)
- **두 기기의 ISQ는 서로 다른 값을 줌** ([근거강함] Bavetta 2024)
- 우리 클리닉은 **Osstell** 사용 (Beacon)

---

## 왜 측정하는가

| 시점 | 측정 목적 |
|---|---|
| 식립 직후 | 1차 안정성(Primary stability) 확인 — 즉시·조기 부하 가능 여부 |
| Healing abutment 시 | 2차 안정성(Secondary stability) 확인 — 골유착 진행 평가 |
| 보철 전 | 부하 시점 최종 결정 |

> ISQ가 **임플란트 실패를 예측하는 단일 최선의 객관 지표**는 아니지만, 다른 임상 지표(PD·BoP·mobility)와 묶어 보면 **부하 결정**의 안전장치 역할.

# Part 2 — 측정 절차 체크리스트

---

## 측정 전 (위생사 준비)

1. SmartPeg (Osstell) 일회용 멸균 확인 — 환자별 1개
2. Beacon 배터리 / sensor head 청결 확인
3. Implant 위치·종류·플랫폼 사이즈 차트에서 확인 → SmartPeg type 선택
4. 환자 동의 / 측정 의미 1문장 설명

---

## 측정 중

1. SmartPeg를 임플란트 fixture/abutment에 finger-tight (≤10 Ncm) 부착
2. Beacon sensor head를 SmartPeg에 직각으로 ~5 mm 거리 유지
3. **4방향 측정** — 협·설·근·원 (M·D·B·L)
4. 각 방향 비프음 후 수치 기록 — 차트 ISQ 필드에 4개 값 기록
5. SmartPeg 회수·폐기 (재사용 금지)

---

## 측정 후 — 차트 기록

```
ISQ (방향): M / D / B / L
평균 ISQ:    (4개 평균)
측정 기기:   Osstell Beacon
SmartPeg #:  type 코드
측정자:      위생사 이니셜
시점:        식립 직후 / 3개월 / 보철 전
원장 확인:   서명
```

> **4방향 중 1개라도 평균과 ≥10 차이 나면 재측정** + 원장 보고.

<!-- _class: lead -->

# Part 3 — 시점별 ISQ 임상 의미

---

## 시점별 임계값 (우리 클리닉 SOP)

| 시점 | ISQ ≥70 | ISQ 65–69 | ISQ <65 |
|---|---|---|---|
| 식립 직후 | 즉시/조기 부하 후보 | 표준 healing(3 mo) | **원장 즉시 보고** — 부하 보류 |
| Healing abutment | 보철 진행 | 4–6주 추가 healing | 원장 보고 — CBCT 재평가 |
| 보철 전 최종 | 부하 OK | 조건부 부하 (보철 디자인 검토) | 부하 금지 — 원장 결정 |

[합의수준] Andersson 2019·국제 가이드라인 기반; [claude해석] 65 cutoff는 ISQ <65 환자의 case 1(좌측 #15) 실패 사례 학습.

---

## 4방향 ISQ 해석 팁

- 4값이 모두 비슷(±5 이내) → 안정적 osseointegration 진행 중
- 1방향만 크게 낮음 → 그 방향 골 결손 의심 → 원장 보고
- 시간 경과에 따른 **상승**(식립 직후 65 → 3개월 72)이 정상 패턴
- **하강**(72 → 68)은 abnormal — 즉시 원장 보고

# Part 4 — DBBM 이식재 표준

---

## DBBM이란

- **Deproteinized Bovine Bone Mineral** = 탈단백 우골 이식재
- 대표: **Bio-Oss** (Geistlich, Swiss) — 세계 최다 사용·연구
- 인체 11년 후에도 입자 안정 ([근거강함] Mordenfeld 2010)
- MSFA 80:20 DPBB+자가골 10년 CSR 86% ([근거강함] Mordenfeld 2014)

---

## 시장 제품 비교 (Manfro 2013 동물실험)

| 제품 | 12주 신생골 (mm²) | 우리 클리닉 사용 |
|---|---|---|
| **Bio-Oss** | 9.32 | **1차 표준** |
| Bone-Fill | 9.01 (≈ Bio-Oss) | 파일럿 5 케이스(발치와) |
| Gen-Ox | 3.02 (1/3) | **사용 중단** |
| 혈병 단독 | 0.63 | (대조군) |

→ "이종골은 다 같다"는 가정 X. **제품명 확인 필수**.

---

## 새로운 옵션 — Bio-Oss + 황산칼슘 hybrid

- Signoriello 2025 후향 (n=24): Bio-Oss + CalMatrix 50:50 vs Bio-Oss 단독
- **생활골 62.5% vs 31.25%** (4개월)
- **무세포골 5% vs 32.91%** (둘 다 p<0.05)
- 우리 클리닉: **3 케이스 한정 파일럿** (발치와 보존, 환자 동의 후)

---

## 우리 클리닉 DBBM 선택 알고리즘

```
   임상 시나리오
   ↓
┌── MSFA / 수직 증대 ──→ Bio-Oss 80% + 자가골 20% (Mordenfeld 2014)
├── 발치와 보존 (4벽 intact) ──→ Bio-Oss 단독 (또는 파일럿: Bone-Fill or Bio-Oss+CaSO₄)
├── 발치와 보존 (벽 결손) ──→ Bio-Oss + Bio-Gide (Bartee 변형)
└── 즉시식립 + buccal gap ≥3 mm ──→ Bio-Oss particulate
```

위생사 역할: **이식재 종류·로트 차트 기록** + **재고 알림**.

# Part 5 — 시나리오 walk-through

---

## 시나리오 A — 식립 직후 ISQ 62 (Case 1 학습)

상황: #15 MSFA + 즉시식립, ISQ 평균 58.

위생사 액션:
1. 4방향 재측정 (calibration 확인)
2. 평균 65 미만 → **원장 즉시 보고**
3. 차트 빨강 flag → 환자 부하 보류 사실 차트 메모

원장 결정 트리:
- ISQ <65 + Bone type IV → **6개월 healing 후 재평가**
- 보철 진행 절대 금지

→ 본 케이스 학습 결과 SOP 강화 ([[note-meeting/2026-05-25_clinical-team-case-conference]] Case 1)

---

## 시나리오 B — 발치와 보존, 4벽 intact

환자: 60세 여, #36 발치 후 임플란트 6개월 후 식립 예정.

SOP:
1. Atraumatic extraction (forceps + periotome)
2. 소켓 curettage·생리식염수 세척
3. **Bio-Oss 단독** 충전 (vs 파일럿: Bone-Fill or Bio-Oss+CaSO₄ — 환자 동의 시)
4. Bio-Gide 콜라겐막 cover
5. 1차 봉합 (coronally advanced flap)
6. 위생사: 이식재 종류·로트 번호 차트 기록 + 6개월 재내원 예약

---

## 시나리오 C — Bone-Fill 파일럿 사용

상황: Bone-Fill 5 케이스 중 첫 케이스.

위생사 액션:
1. 환자 사전 동의서 별도 확인 (파일럿 제품 사용)
2. 차트 "PILOT: Bone-Fill #1/5" 명시
3. 6개월 재내원 시 임플란트 식립 시 추가 trephine biopsy 채취 동의 확인
4. 결과 데이터: 원장이 2026-12 케이스 컨퍼런스 발표
