## One-line Summary

In-vitro fracture-resistance study (90 mandibular premolars, 9 groups × 10): pooled FRC-post groups had significantly higher fracture load than ParaPost groups (380.1 vs 262.8 N, P<0.05), with no effect of core material — closely replicating Izadi et al. 2020.

## Summary

Goutam et al. (2022) evaluated fracture resistance of endodontically treated mandibular premolars restored with fiber-reinforced composite (FRC) posts versus titanium **ParaPosts**, across three core build-up materials (Photo Core, LuxaCore, Core Max II). Pooled FRC-post groups (380.1 N) significantly exceeded pooled ParaPost groups (262.8 N, P<0.05), and core material type had no effect on fracture resistance. The design, group structure, loading protocol (Panavia F2, 45°, 1 mm/min), and even the reported group means are strikingly close to Izadi et al. 2020 (which this paper cites) — so the two should be read as one finding independently reproduced rather than two fully independent datasets. Its evidential contribution is confirmatory: fiber > metal (ParaPost) fracture load, core material irrelevant.

## Key Contributions

- Independent (India-based) replication of the FRC-post-vs-ParaPost fracture comparison.
- Confirms FRC > ParaPost pooled fracture load in a smaller sample (n=90).
- Reconfirms that core material (Photo Core / LuxaCore / Core Max II) is not a determinant.

## Methodology

- In-vitro fracture resistance, universal testing machine. 90 extracted mandibular first premolars, 9 groups (n=10).
- I FRC post; II–V FRC + Photo Core / LuxaCore / composite core / Core Max II; VI ParaPost; VII–IX ParaPost + LuxaCore / Photo Core / Core Max II.
- Root canal treatment; silicone PDL simulation; acrylic mounting; post space with Peeso reamers; Panavia F2 post cement; core build-up; crowns seated.
- Load 45° to long axis, 1 mm/min to fracture. SPSS 21, Student's t + one-way ANOVA (P<0.05).

## Results

| Group | Mean ± SD (N) |
|---|---|
| I FRC post | 452 ± 61.5 |
| II FRC + Photo Core | 412.6 ± 42.4 |
| III FRC + LuxaCore | 362.4 ± 40.8 |
| IV FRC + composite core | 430.4 ± 112.4 |
| V FRC + Core Max II | 384.2 ± 90.6 |
| VI ParaPost | 288.1 ± 81.5 |
| VII ParaPost + LuxaCore | 240.6 ± 74.2 |
| VIII ParaPost + Photo Core | 276.4 ± 68.1 |
| IX ParaPost + Core Max II | 246.5 ± 50.2 |

- Pooled FRC (II–V) 380.1 ± 72.1 vs pooled ParaPost (VI–IX) 262.8 ± 70.6; t=8.2, **P=0.01**.
- Core material type: no significant effect.
