## One-line Summary

In-vitro fracture-resistance study (108 mandibular premolars, 9 groups × 12): pooled FRC-post groups had significantly higher fracture load than ParaPost groups (397.6 vs 265.6 N, P<0.001), while the three core build-up materials (Photo Core, LuxaCore, Core Max II) showed no superiority over each other.

## Summary

Izadi et al. (2020) ran a 2×4 factorial in-vitro study (108 mandibular first premolars) to separate two questions: does post type (fiber-reinforced composite vs titanium **ParaPost**, Coltène/Whaledent) matter, and does core build-up material (Photo Core, LuxaCore, Core Max II ± bonding) matter, for fracture resistance of endodontically treated teeth restored under full-metal crowns? Post type mattered strongly: pooled FRC-post groups reached 397.6 N versus 265.6 N for pooled ParaPost groups (P<0.001). Core material did not — no core product was significantly better than another within either post type, and adding a bonding agent to Core Max II gave only a small non-significant gain. The highest single group was FRC + Core Max II with bonding (423.7 N); the lowest was ParaPost + LuxaCore (242.3 N). The clinical implication: the choice between fiber and metal post drives fracture behavior far more than which composite core material is used.

## Key Contributions

- Isolates post-type effect from core-material effect in one 108-tooth factorial design.
- FRC posts significantly outperform titanium ParaPosts on pooled fracture load (P<0.001).
- Core build-up material (Photo Core / LuxaCore / Core Max II ± bonding) is not a determinant of fracture resistance.
- Bonding agent with Core Max II yields a non-significant increase only.

## Methodology

- In-vitro fracture resistance, universal testing machine. 108 sound single-rooted mandibular first premolars, randomized to 9 groups (n=12).
- Group 1 control (intact); Groups 2–5 FRC post + Photo Core / LuxaCore / Core Max II with bonding / without bonding; Groups 6–9 ParaPost + same four core conditions.
- RCT (step-back, ISO 40, gutta-percha + AH26); silicone PDL simulation; acrylic mounting; full-metal crown (glass ionomer cemented).
- ParaPost: titanium vented & serrated No. 4 (Coltène/Whaledent); #2 peeso-reamer then #4 ParaPost drill (12 mm). Panavia F2 post cement. Load 45° to long axis, 1 mm/min to fracture. ANOVA + Tukey (α=0.05).

## Results

| Group | Post + Core | Mean ± SD (N) |
|---|---|---|
| 1 | Control (intact) | 454.0 ± 62.7 |
| 2 | FRC + Photo Core | 410.8 ± 48.3 |
| 3 | FRC + LuxaCore | 365.1 ± 42.1 |
| 4 | FRC + Core Max II (+bond) | 423.7 ± 111.7 (highest post group) |
| 5 | FRC + Core Max II (−bond) | 392.4 ± 90.0 |
| 6 | ParaPost + Photo Core | 292.3 ± 83.9 |
| 7 | ParaPost + LuxaCore | 242.3 ± 73.4 (lowest) |
| 8 | ParaPost + Core Max II (+bond) | 278.2 ± 67.9 |
| 9 | ParaPost + Core Max II (−bond) | 247.3 ± 49.6 |

- Pooled FRC (2–5, n=46) 397.56 ± 77.93 vs pooled ParaPost (6–9, n=45) 265.62 ± 71.05; t=8.4, **P<0.001**.
- Within each post type, core material differences not significant.
