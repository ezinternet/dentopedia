## 한줄요약
근관 성형(canal shaping) in-vitro/ex-vivo 7편 종합: (1) glide path는 수동 SS K-file이 어떤 rotary pathfinder보다 transportation이 크고, rotary 시스템끼리는 동등 — 단 어떤 kinematics도 apical 세균 압출(bacterial extrusion)을 피하지 못함; (2) 본 성형 단계에서 low-taper TruNatomy가 곡관 보존·시간·삭제량에서 ProTaper Gold보다 우수하고, ProTaper Gold는 transportation·fatigue 양쪽에서 비교군 하한; (3) autoclave는 heat-treated NiTi의 cyclic fatigue를 약화시키지 않고 일부는 강화 — 재사용 SOP는 멸균 횟수가 아니라 사용 횟수·곡률로 판단. 전 근거 bench-only, 임상 종점(success·healing·in-vivo fracture·post-op pain) 없음.

## Summary
이 페이지는 `endodontics/shaping` 카테고리의 in-vitro/ex-vivo 7편을 세 축으로 묶는다: glide-path 단계의 기하학적 transportation과 생물학적 비용, 본 성형(main shaping) 단계의 file 선택과 곡관 보존, 그리고 NiTi 합금의 cyclic fatigue와 autoclave 멸균 상호작용. 세 축은 독립 주제처럼 보이지만 하나의 임상 결정 — "곡률이 복잡한 근관에서 어떤 file을, 몇 번까지, 어떤 kinematics로 쓸 것인가" — 로 수렴한다.

[근거강함은 해당 없음] 7편 전부 `confidence: in-vitro`다. 따라서 아래 결론은 bench 수준에서 일관되며, 임상 적용은 모두 [claude해석] 외삽이다.

## Results

### 축 1 — Glide Path Geometry: 수동 vs Rotary
두 독립 연구가 기질을 달리해 같은 결론에 도달한다. Aflaki 2020 (천연 dentin, 하악 대구치 MB, 25–45° 곡률, n=51, CBCT)에서 수동 SS K-file은 3 mm·9 mm 레벨에서 PathFile·Scout RaCe보다 유의하게 큰 transportation을 냈고, 두 rotary는 동등했다. Aminsobhani 2022 (S-shaped resin block, n=100, 5군)는 이를 확장 — 수동 K-file이 apical·middle third에서 가장 큰 deviation (p<0.001), Scout RaCe·One G·PathFile·GPS 4종 rotary는 서로 통계적 차이 없음.

함의: glide-path rotary pathfinder 간 transportation 차이는 무시 가능 → 선택 기준은 비용·가용성. 수동 K-file은 곡관에서 1차 선택에서 배제. [합의수준] (bench 일관성 기반)

### 축 2 — Glide Path의 생물학적 비용
Pui-Yii 2021 (발치 단근치, E. faecalis 접종)은 기하 연구가 무시하는 차원을 측정한다: 수동 K-file·One G 연속회전·WaveOne Gold Glider 왕복 — 세 kinematics 모두 apical foramen 너머로 측정 가능한 세균을 압출했다. 즉 rotary로 transportation을 줄여도 생물학적 압출은 0이 되지 않는다. flare-up 위험군에서는 file 선택과 무관하게 apical patency 관리와 압출 최소화 술기가 필요. [claude해석]

### 축 3 — 본 성형 file의 곡관 보존
Kim 2021 (S-shaped resin, n=60, PTG/WaveOne Gold/TruNatomy)에서 minimal-taper TruNatomy가 3·5 mm 레벨 transportation 최소, 삭제량 최소, 시간 최단. ProTaper Gold는 apical curve(2–3 mm)에서 transportation 최대. 곡률이 복잡할수록 taper가 낮은 시스템이 곡관 중심을 유지한다는 방향. [합의수준]

### 축 4 — NiTi Cyclic Fatigue & 곡률 stressor
ProTaper Gold는 여러 연구에서 "비교군 하한" 역할을 한다. Khalil 2019 (single vs double curvature, n=15/group)에서 EdgeEvolve가 PTG보다 cyclic fatigue·굴곡 유연성·표면 평활도(AFM Ra) 모두 우수 — Ni/Ti 조성은 동일한데도. 또한 double-curvature 근관은 두 시스템 모두 fatigue resistance를 낮췄다(p<0.05). Almohareb 2023에서도 EdgeTaper Platinum > ProTaper Gold ≈ TruNatomy. 정리하면 곡률 복잡도(double curve·S-shape)가 file 수명을 결정하는 1차 stressor이고, file 간 차이는 단순 근관에서 희미해지고 복잡 근관에서 증폭된다. [합의수준]

### 축 5 — Autoclave 멸균은 NiTi를 약화시키지 않는다
Özyürek 2017 (ProTaper Universal/Next/Gold, n=240, 4 멸균 프로토콜): baseline에서 PTG > PTU·PTN, 그리고 10회 pre-test autoclave가 PTN·PTG의 number of cycles to failure(NCF)를 유의하게 증가. Almohareb 2023: EdgeTaper Platinum은 1회·10회 autoclave 후 NCF 유의 증가, ProTaper Gold·TruNatomy는 멸균 중립(neutral). 방향: heat-treated NiTi의 routine autoclave는 fatigue 손실 없음, 일부는 합금 aging(R-phase 재배열 추정)으로 강화. [근거강함] (두 독립 trial 수렴)

**불일치 플래그**: ProTaper Gold가 Özyürek 2017에서는 autoclave로 NCF↑, Almohareb 2023에서는 중립. 원인 후보 — file size/taper 차이(Almohareb는 PTG 25/.08), cycle 수(1/5/10 vs 10), 시험 canal 기하 차이. 동일 명칭 file이라도 단면·테이퍼가 다르면 결과가 갈릴 수 있어, "PTG는 autoclave로 강화된다"고 단정 불가. [claude해석]

## 임상 의사결정 ladder (canal shaping)

| 단계 | 권장 | 대안 / trade-off |
|---|---|---|
| Glide path (곡관) | rotary pathfinder 1종(비용 기준 선택) — transportation 동등 | 수동 K-file은 곡관에서 transportation 최대, 직선·넓은 근관에서만 허용. WaveOne Gold Glider 왕복은 단일 file 효율 ↔ 압출은 여전 |
| 본 성형 (S-shape·double curve) | low-taper 우선 — TruNatomy(곡관 보존·삭제 최소·시간 최단) | ProTaper Gold는 transportation 하한이나 가용성·익숙함 이점. EdgeEvolve/EdgeTaper Platinum은 fatigue·유연성 우위 |
| File 재사용 폐기 기준 | 사용 횟수 + 곡률 노출(double curve 1회=큰 stress)로 판단 | 멸균 횟수는 기준 아님 — autoclave는 fatigue 비악화·일부 강화 |
| 멸균 SOP | heat-treated NiTi는 표준 autoclave 그대로 | 멸균이 fatigue를 깎는다는 우려는 bench 근거상 기각 |

최종 판단은 원장 몫. 위 ladder는 bench 근거의 외삽이며 임상 종점(치유율·in-vivo 파절·술후 통증) 직접 근거는 본 7편에 없음.

## 근거 한계 (전 축 공통)
- 7편 모두 in-vitro/ex-vivo. RCT·코호트·임상 종점 0건.
- resin block 의존(Aminsobhani·Kim·Khalil) — dentin 미세경도 미반영; Aflaki만 천연 dentin.
- 단일 술자·단일 기하 설계가 대부분.
- Pui-Yii는 단일 균종(E. faecalis)·단근치 — 다근치 대구치 multi-species biofilm 미모델.
- in-vivo 파절률·flare-up 연결 데이터 없음.
