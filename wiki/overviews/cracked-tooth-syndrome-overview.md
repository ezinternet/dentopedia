## 한줄요약
균열치 증후군 (Cracked Tooth Syndrome, CTS) 신규 8편 통합 — (1) longitudinal fracture 5-type 분류, (2) 진단 multi-modal framework (visual·bite test·transillumination·CBCT·QLF), (3) 치료 결정의 핵심 tension — Kaur 2026 umbrella "endo + full cuspal coverage 최고 prognosis" vs Kwak 2024 "unnecessary endo 회피" 보수적 입장. 한국 임상 적용 framework 제시.

## Scope
2026-05-27 이전 wiki에 없던 CTS 카테고리를 신설하면서 동시 ingest한 8편의 종합. 4편의 international cluster (Kaur 2026 umbrella + Ferracane 2023 PBRN cohort + Li 2021 종합 review + Raj 2025 mini review)와 4편의 JKDA 한국 임상 cluster (Kim & Shin 진단·Son QLF·Kwak endo 결정 + Kaur 2026 protocol)가 짝을 이룬다. CTS 분야는 primary RCT가 빈약하고 SR/cohort/review 기반 — 본 overview는 그 evidence base의 한국 임상의용 의사결정 지도.

## Cluster 1 — 분류 framework (Longitudinal Fracture 5-Type)

[[cracked-tooth/raj-2025-cracked-tooth-syndrome-diagnostic-dilemma]] (FROH 2025, mini review)는 CTS가 단독 진단명이 아니라 **longitudinal fracture 5 type 중 incomplete fracture에 해당**한다는 분류 framework을 명확히 한다:

| Type | 특징 | 치료 분기 |
|---|---|---|
| **Craze line** | 표면 enamel only, 무증상 | 관찰 |
| **Fractured cusp** | 교두 partial 파절 | 보존적 restoration ± crown |
| **Cracked tooth (CTS)** | Incomplete fracture, pulp까지 진행 가능 | **본 overview 핵심** |
| **Split tooth** | Complete fracture, 치아 둘로 분리 | 부분 발치 또는 전체 발치 |
| **Vertical root fracture** | 치근 종방향 파절 | 보통 발치 |

[[cracked-tooth/li-2021-cracked-tooth-syndrome-etiology-review]] (PRM 2021)은 etiology를 **iatrogenic vs non-iatrogenic**로 구분 — 의료 행위 관련 (부적절 endo·deep cavity preparation·post 시술)과 비의료 (교합력·이갈이·외상·치아 구조 약화). [근거강함] 본 분류는 분야 통용 standard이지만, iatrogenic 강조는 시술자에게 self-reflection 기회를 제공한다 — overtreatment·invasive 술식 경계가 분야 임상 문화의 한 축이다.

**Synthesis [claude해석]:** CTS와 split tooth의 경계는 임상에서 시간 경과로 결정된다 — early intervention이 split 진행 차단의 핵심이다. Craze line이 CTS로, CTS가 split으로 진행하는 dynamic을 환자에게 설명할 framework이 본 분류의 임상 가치다.

## Cluster 2 — 진단의 multi-modal framework

[[cracked-tooth/jkda-2024-62-1-003]] (Kim & Shin, JKDA 2024, 균열치 특집 1번)이 핵심 메시지를 명확히 한다: **single tool로 확진 불가**. 임상 평가 + 영상 + 환자 호소를 종합한 multi-modal framework이 필수.

### 도구별 정확도·한계

| 도구 | 장점 | 한계 |
|---|---|---|
| Visual + transillumination | 빠름, 비침습 | 깊은 crack 누락 |
| Dye staining (methylene blue) | 시각화 | 시술 침습 |
| Bite test (Tooth Slooth) | 환자 호소 재현 | 위양성·위음성 |
| **CBCT** | 3D | Crack은 직접 시각화 어려움 |
| **QLF** ([[cracked-tooth/jkda-2024-62-1-004]]) | 비침습·정량적·405nm 청색광 형광 | 장비 가용성·비용, 적용 RCT 부족 |
| 진단 결정 검사 (endo 진입) | 직접 확인 | 비가역적 |

[[cracked-tooth/jkda-2024-62-1-004]] (Son, JKDA 2024)는 QLF의 활용을 보완 도구로 제시하지만 균열치 specific RCT는 부족 — 본 overview는 QLF를 1차 screen 또는 기존 도구의 보완으로 위치시킨다. [합의수준]

### 권장 진단 sequence (한국 임상 적용)

1. Patient history: sharp pain on chewing release (CTS 특이 호소)
2. Visual + transillumination
3. Bite test (cusp별 sequential — 어느 cusp 작용 시 통증인가)
4. CBCT (deep crack·root involvement 의심 시)
5. (선택) QLF screen
6. Tentative diagnosis → endo 진입 후 final confirmation

**Synthesis [claude해석]:** [[cracked-tooth/raj-2025-cracked-tooth-syndrome-diagnostic-dilemma]]가 "diagnostic dilemma"로 부르는 이유는 어떤 단일 도구도 sensitivity와 specificity를 동시에 보장 못 하기 때문이다. 한국 임상에서 외래 시간 제약을 고려하면 history + visual + bite test가 minimal default이고, CBCT는 dimensional/spatial 진단 결정 시점에 추가하는 것이 현실적이다.

## Cluster 3 — 치료 결정의 핵심 tension

CTS overview의 가장 임상적으로 중요한 부분 — **endo 진입 여부와 cuspal coverage 결정**. 두 입장이 정면 대립한다.

### 입장 A: Endo + full cuspal coverage 적극 권장

[[cracked-tooth/kaur-2026-cracked-tooth-syndrome-umbrella-review]] (JCDE 2026, PROSPERO CRD420250648720, 4 SR / 30 primary studies / CCA 17%):
- **Endodontic therapy 후 full cuspal coverage가 최고 prognosis** 결론
- Endo 없이 crown only 또는 endo 없이 partial coverage 모두 열등
- AMSTAR-2 평가·CCA 정량 — methodological transparency

[[cracked-tooth/ferracane-2023-cracked-tooth-registry-3year-network]] (National Dental PBRN, 8 articles 2017-2022 lessons learned, OHSU):
- 후방 cracked teeth 3년 multi-site cohort
- 3 핵심 임상 질문 답:
  - **Q1: Which cracked teeth will get worse?** Deep crack into dentin/pulp, symptomatic, pulp involvement
  - **Q2: When should practitioners intervene?** Spontaneous pain, periapical pathology, deep crack with pulp risk, cuspal mobility
  - **Q3: What is the best treatment?** **Cuspal coverage (full crown / onlay) 강력 권장** — Kaur 2026 umbrella와 일관

[근거강함] 두 연구는 분야 highest evidence와 largest real-world cohort — **국제 표준은 endo + crown 적극**

### 입장 B: 보수적 — unnecessary endo 회피

[[endodontics/kwak-2024-endodontic-treatment-cracked-tooth-management]] (Kwak, JKDA 2024, Yonsei Nature 치과의원, 균열치 특집 3번):
- **균열치 발견 = 자동 endo 아니다** 강조
- 많은 case는 endo 없이 cuspal coverage·crown만으로 해결
- Endo trigger 명확화: spontaneous pain, periapical 반응, deep crack with pulp 침범
- 모호 case: temporary crown + 4-8주 관찰 → 재평가
- **Pulp 생명 보존 우선**

[claude해석] Private practice perspective — overtreatment 경계가 핵심 메시지

### 결정 트리 (양 입장 통합)

| 임상 시나리오 | Kaur/Ferracane 권장 | Kwak 권장 | 통합 권장 |
|---|---|---|---|
| 자발통, periapical pathology | Endo + crown | Endo | **Endo + crown** (일치) |
| Pulpitis 진행, deep crack | Endo + crown | Endo + crown | **Endo + crown** (일치) |
| Bite-only pain, vital pulp, shallow crack | Cuspal coverage (full crown) | Conservative restoration + observation | **갈라짐** — 환자 선호·risk tolerance |
| Cuspal mobility | Endo + crown | Crown 우선, endo 모호 시 보류 | **Cuspal coverage 최소** — endo는 case-by-case |
| 모호 통증, vital pulp | Cuspal coverage | Temporary crown + 4-8주 관찰 | **임시 처치 후 재평가** (Kwak 입장 우위) |

**Synthesis [claude해석]:** 두 입장의 차이는 sample population 차이로 부분 설명 가능 — Kaur umbrella·Ferracane PBRN cohort는 전문진료센터·academic 환경 (이미 deep crack·symptomatic case 위주)인 반면, Kwak은 일반 개원 환경 (early/mild case 비율 높음). 한국 일차 임상 (개원의)은 Kwak의 보수적 framework이 default로 합리적이지만, 결정 trigger (자발통·periapical pathology·cuspal mobility) 발생 시 즉시 Kaur/Ferracane 입장으로 전환해야 한다.

[합의수준] Cuspal coverage (full crown 또는 onlay)는 양 입장 모두 동의 — endo 여부와 별개로 cuspal coverage는 standard.

## 한국 임상 적용 framework (Practical Workflow)

### 외래 1회차 (진단)
1. History — "씹을 때 짧고 날카로운 통증, 풀릴 때 더 아픈가?" (CTS pathognomonic)
2. Visual + transillumination
3. Bite test (cusp별 sequential)
4. 잠정 진단 → cuspal coverage 필요성 평가

### 외래 2회차 (분기 결정)
| 상황 | 결정 |
|---|---|
| 무자발통 + bite-only pain + vital pulp | **Temporary crown + 4-8주 관찰** (Kwak) |
| 자발통 + periapical pathology | **Endo + crown** (Kaur/Ferracane) |
| Pulp 침범 의심 | CBCT + endo 진입 결정 |
| 진단 모호 | 4-8주 관찰 후 재평가 |

### 외래 3회차+ (재평가)
- 통증 개선 → cuspal coverage (onlay or crown) 진행, endo 보류
- 통증 지속·악화 → endo + crown
- 새로운 자발통·periapical pathology → endo + crown

### 환자 설명·동의서 framework
- "Crack은 시간이 지나면 진행할 수 있음" (split tooth로 진행 risk)
- "교두 덮음 (crown/onlay)이 진행 차단의 핵심"
- "신경치료 결정은 case-by-case — 자발통·X-ray 변화 시 추가 결정"
- "정기 점검 필수" (Ferracane 3-year cohort 기반)

## Strengths and Gaps

### 강점
- [근거강함] Kaur 2026 umbrella + Ferracane 2023 cohort는 분야 best evidence
- [합의수준] Cuspal coverage 권고는 양 입장 모두 동의
- [근거강함] Raj 2025의 longitudinal fracture 5-type 분류 standard

### 약점·gap
- [미검증] CTS primary RCT 절대 빈약 — umbrella·SR도 결국 동일 base에 의존
- [claude해석] Kaur 2026 CCA 17%는 primary study 중복 사용 — true heterogeneity 평가 한계
- [미검증] QLF의 균열치 specific RCT 부족 — Son 2024 review는 가능성 제시 단계
- [미검증] Long-term (≥5년) cohort 부족 — Ferracane 3년이 가장 큼
- [claude해석] 한국 임상 분포 데이터 (외래 CTS 비율·환자 특성) 없음 — 본 overview는 국제 cohort 기반 framework의 한국 적용 시도

### 향후 보강 필요
- Korean cohort CTS 임상 분포 데이터
- QLF + CBCT 통합 진단 RCT
- Conservative restoration vs cuspal coverage RCT (non-symptomatic case)
- 5년+ long-term outcome

## Bottom Line (한 줄 결론)

[근거강함] CTS = longitudinal fracture 5 type 중 incomplete fracture. Multi-modal 진단이 standard (single tool 불가). 치료의 핵심 question = **endo 여부**이고 cuspal coverage는 모든 입장에서 standard.

[합의수준] **국제 SR (Kaur 2026·Ferracane 2023)은 endo + crown 적극 권장**, **한국 JKDA (Kwak 2024)은 unnecessary endo 회피 보수적 입장** — sample population 차이로 부분 설명 가능. 임상 권장은 한국 일차 임상에서 Kwak default + 결정 trigger 발생 시 Kaur/Ferracane 입장 전환.

[claude해석] 환자 입장에서는 "교두 덮음은 필수, 신경치료는 case-by-case 결정" framework이 가장 정확한 설명. CTS 분야는 RCT가 빈약해서 임상의의 judgment가 중요 — 결정 trigger를 명확히 (자발통·periapical pathology·cuspal mobility) 정해두고 정기 추적이 best practice.
