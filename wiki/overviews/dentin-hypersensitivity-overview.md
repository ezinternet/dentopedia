## 한줄요약
시린이 (Dentin Hypersensitivity, DH) 통합 overview — JKDA 2022 특집 3편 + 인접 NCCL (Noncarious Cervical Lesion) 카테고리 통합. (1) Hydrodynamic theory 기반 메커니즘, (2) 외래 진단·치료 ladder (at-home → in-office sealing → restorative), (3) iatrogenic DH 예방 (Class II direct composite postop + NCCL restoration). 한국 임상 외래 의사결정 framework.

## Scope
DH는 한국 임상에서 흔한 호소 (전세계 유병률 8-57%) 이지만 single tool 진단 불가·치료 옵션 다양함이 임상 의사결정의 challenge. 본 overview는 JKDA 2022 특집 (Jeon&Seo overview, Jang clinical management, Kim postop prevention)을 axis로 [[resin-bonding/]] NCCL cluster와 [[endodontics/]] pulp 관련 페이지를 연결한다. DH 분야는 RCT가 부분적이고 narrative review 중심 — 본 overview는 임상 적용 framework 정리.

## Cluster 1 — Mechanism (Hydrodynamic Theory)

[[dentin-hypersensitivity/jkda-2022-60-4-003]] (Jeon & Seo, JKDA 2022 특집 1번)은 DH 메커니즘을 **Brännström의 hydrodynamic theory**로 정리한다:

```
노출된 상아질 (Exposed Dentin)
       ↓
상아질 세관 (Dentinal Tubule) 노출
       ↓
열·삼투·기계·증발 자극
       ↓
관내 유체 이동 (Fluid Movement)
       ↓
Odontoblast / pulp 신경 자극
       ↓
짧고 날카로운 통증 (Sharp Brief Pain)
```

**Synthesis [claude해석]:** Hydrodynamic theory는 단일 가설이지만 분야 합의에 가깝다. 임상 의사결정의 본질적 결과 — **치료의 두 축은 "관 폐쇄 (tubule occlusion)" 또는 "신경 탈민감화 (nerve desensitization)"** — 본 이론에서 자연 도출된다. 모든 treatment option (KNO3·NovaMin·arginine·varnish·laser·resin sealant)은 이 두 축 중 하나 또는 둘에 작용한다. [근거강함]

### DH vs 균열치 (cracked tooth) 감별

| 특징 | DH | Cracked Tooth |
|---|---|---|
| 통증 유발 | 자극 (cold·sweet·brushing) | 씹기 (특히 풀릴 때 sharper) |
| Bite test | (-) 또는 mild | (+) — 특정 cusp 작용 시 |
| 자발통 | 없음 | 진행 시 가능 |
| 영상 | 보통 정상 (NCCL 동반 가능) | 보통 정상 (CBCT도 직접 시각화 어려움) |
| 위치 | 치경부·교합면 | 특정 cusp 또는 전체 치관 |

[claude해석] DH와 cracked tooth가 동반될 수 있음 — 환자가 "씹을 때 시리다"고 하면 [[overviews/cracked-tooth-syndrome-overview]] framework 병행 검토 필요.

## Cluster 2 — Treatment Ladder (외래 의사결정)

[[dentin-hypersensitivity/jkda-2022-60-4-004]] (Jang, JKDA 2022 특집 2번)은 외래 의사결정 ladder를 단계별로 제시한다:

### 단계별 옵션 매핑

| 단계 | 옵션 | 작용 기전 | 적응 |
|---|---|---|---|
| **1차 (at-home)** | KNO3 (질산칼륨) toothpaste | 신경 탈민감화 | 경증, conservative first |
| | NovaMin (calcium sodium phosphosilicate) | 관 폐쇄 + 재석회화 | 경증·중등도 |
| | Arginine (Pro-Argin) | 관 폐쇄 | 경증·중등도 |
| **2차 (in-office, sealing)** | Fluoride varnish | 관 폐쇄 (저강도) | 경증·중등도, [[caries/jkda-2023-61-7-004]] 참조 |
| | Oxalate (e.g., BisBlock) | 관 폐쇄 (인산칼슘 결정) | 중등도 |
| | Glutaraldehyde + HEMA (Gluma) | 관 단백질 응고 + 폐쇄 | 중등도 |
| | Laser (Nd:YAG, diode, Er:YAG) | 관 폐쇄 (열·resonance) | 중등도·중증, 자가요법 무효 |
| | Resin sealant | 관 폐쇄 (장벽 형성) | 중등도·중증 |
| **3차 (restorative)** | Composite restoration | 결손 회복 + 관 폐쇄 | 깊은 NCCL, 보존적 무효 |
| | Crown / onlay | 전체 결손 회복 | 광범위 결손 |

### 환자 행동 수정 (양 단계 동반)
- 브러싱 기법: scrubbing → modified Bass 또는 Stillman (압력·각도 조절)
- Acidic diet 회피: citrus, vinegar, carbonated drinks
- Bruxism guard (이갈이 동반 시) — [[botulinum-toxin/jkda-2025-63-6-002]] 참조

**Synthesis [claude해석]:** Ladder의 핵심은 "**conservative first, escalate only if fails**". 한국 임상에서 환자가 처음 외래 방문 시 자가요법 (KNO3 toothpaste·올바른 브러싱 교육) 4-8주 → 재평가가 default. 즉시 laser나 restoration으로 가는 것은 overtreatment risk. 단, 환자가 자가요법 이미 시도했거나 결손 (NCCL 깊은) 동반 시 2차/3차로 직진 가능.

[합의수준] 분야 전반의 consensus framework. 단 각 옵션의 head-to-head RCT는 부분적 — 환자 선호·접근성·비용도 결정 인자.

## Cluster 3 — Iatrogenic DH 예방 (시술 후 시린이)

DH는 환자의 자연 발생만이 아니라 **수복 후 발생 (postoperative hypersensitivity)**도 중요한 임상 issue. [[dentin-hypersensitivity/jkda-2022-60-4-005]] (Kim, JKDA 2022 특집 3번)이 Class II direct composite 수복에서 예방 전략을 제시한다.

### Class II direct composite 수복 후 DH 예방 protocol

| 단계 | 권장 | 메커니즘 |
|---|---|---|
| **Adhesive technique** | Selective etching·adequate primer 적용 | Microleakage 차단 |
| **Filling** | Incremental (얇은 layer 분할) | Polymerization shrinkage 분산 |
| **Curing** | Slow-start (낮은 출력 시작 → 증가) | Shrinkage stress 감소 |
| **Proximal box** | Pre-wedging + sectional matrix system | 적절한 contact + 변연 적합 |

### NCCL 수복에서 DH/postop sensitivity (인접 cluster)

[[resin-bonding/assis-2023-acid-etching-universal-adhesive-nccl-sr-ma]] (SR+MA 20 RCT)은 universal adhesive의 etch-and-rinse (E&R) vs self-etch (SE) trade-off를 정량:

- **E&R 우월**: 중기 (12-36개월) 유지율, 변연 적합·착색, 2차 우식
- **SE 우월**: **술후 sensitivity 낮음**

→ NCCL 수복 시 long-term 결과와 postop sensitivity 사이 trade-off. 환자가 sensitivity 경향이 있거나 NCCL이 깊지 않으면 SE 우선 고려. [근거강함]

[[resin-bonding/oza-2022-universal-adhesive-nccl-rct-2year]] · [[resin-bonding/doshi-2023-universal-adhesive-nccl-strategy-sr-ma]] 도 같은 framework 보강.

### iatrogenic DH의 핵심 원인 (요약)
1. **Polymerization shrinkage** → microleakage → 관내 유체 이동
2. **Inadequate adhesive** → 변연 누출
3. **Bulk filling without slow curing** → 응력 집중
4. **Insufficient proximal contact** → 음식 잔류·자극
5. **Over-etching of dentin** (E&R 선택 시) → 신경 자극

**Synthesis [claude해석]:** Kim 2022 framework은 표준이지만 bulk fill resin 발달로 incremental 필요성 일부 감소. 핵심은 **adhesive technique과 slow curing** — bulk fill을 쓰더라도 이 두 축은 유효. NCCL 수복 시 SE adhesive가 postop sensitivity 측면에서 안전한 default.

## 한국 임상 적용 framework (Practical Workflow)

### 외래 1회차 (진단·1차 처치)
1. History — "어떤 자극에 시린가? (cold·sweet·brushing·biting·spontaneous)"
2. Visual: NCCL 유무, 결손 깊이, 노출 dentin 부위
3. Bite test → cracked tooth 감별
4. 잠정 진단:
   - **DH only** (자극성 통증, NCCL 미경증) → at-home 1차 (KNO3 toothpaste + 브러싱 교육)
   - **DH + NCCL 깊음** → 2차 (varnish or restoration) 즉시 진입
   - **DH + cracked tooth 의심** → [[overviews/cracked-tooth-syndrome-overview]] framework 병행

### 외래 2회차 (4-8주 재평가)
| 결과 | 결정 |
|---|---|
| 개선 → 통증 없음 | OHI maintain, 정기 점검 |
| 부분 개선 | At-home 계속 + 2차 in-office (varnish, oxalate) 추가 |
| 무개선 또는 악화 | 2차/3차 escalate (laser, resin sealant, composite restoration) |
| 새로운 자발통 | DH 재평가, cracked tooth/pulpitis 감별 |

### 환자 설명·동의서 framework
- "시린이는 노출된 상아질 세관 내 유체 이동으로 발생"
- "치약·올바른 브러싱·식이 조절이 1차"
- "자가요법 4-8주 후 효과 없으면 진료실 치료 (varnish·laser·수복)"
- "이갈이 있으면 보호 장치 동반 필요"
- "수복 (composite) 후 일시적 시린이 가능 — 1-2주 내 호전"

### Hygienist 매뉴얼 체크리스트 (외래 hygienist 역할)
1. 환자 통증 ladder (1-10) 기록
2. NCCL 유무·깊이 visual chart
3. 브러싱 시범 (modified Bass) 교육
4. KNO3 toothpaste 처방 (브랜드명 명시)
5. Acidic diet 회피 list 제공 (인쇄물)
6. 4-8주 재내원 예약

## Bottom Line (한 줄 결론)

[근거강함] DH 메커니즘은 hydrodynamic theory (관내 유체 이동) — 모든 treatment의 두 축은 **관 폐쇄** 또는 **신경 탈민감화**.

[합의수준] 외래 ladder는 **at-home (KNO3·NovaMin·arginine) 4-8주 → in-office sealing (varnish·oxalate·laser·resin) → restorative**. Conservative first가 default, escalate only if fails.

[claude해석] iatrogenic DH (Class II composite postop, NCCL restoration) 예방은 **adhesive technique + slow curing**이 핵심. NCCL adhesive 선택은 long-term (E&R 우위) vs postop sensitivity (SE 우위) trade-off — 환자 sensitivity 경향 시 SE 우선. Cracked tooth와 동반 case는 [[overviews/cracked-tooth-syndrome-overview]] framework 병행 필수.
