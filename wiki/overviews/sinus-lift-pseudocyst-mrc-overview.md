## 한줄요약
상악동 점액낭종 (Antral Pseudocyst, AP)·점액저류낭종 (Mucous Retention Cyst, MRC) 동반 상악동거상술 + 임플란트 의사결정의 통합 framework — 15편 (1 SR + 1 RCT + 4 cohort + 7 case + 2 review) 종합. **4단계 결정 트리**: (0) 진단·감별 → (1) 위험 자체 평가 (Anitua 2021 SR: 임상적 유의 위험 미입증) → (2) 처치 옵션 결정 (제거·흡인·미처치 중 subtype·크기 기반) → (3) 장기 trajectory 모니터링. 1년 (Berberi)·6년 (Silva)·15년 (Grün) datapoint로 trajectory anchor.

## Scope
2026-05-27 ingest로 신설된 [[sinus-lift/pseudocyst/]] 카테고리 15편 전체 통합. SR + RCT + 후향 cohort (n=26·107) + 6 case series + 2 review로 evidence pyramid의 전 layer를 포괄. 분야 핵심 question은 **"낭종 있는데 sinus lift + implant 해도 되는가, 어떻게 처치할 것인가"** — 본 overview는 한국 임상의 결정 트리 step-by-step.

분야 일반 review (Anitua 2021 SR)와 한국 임상 case series (jkaoms 원광대 OMFS, jkda 전북대 치주과, scs 서울대 OMFS)를 짝지어 한국 임상 적용 framework 제시.

## Cluster 0 — 진단·감별 (Step 0: 정확한 진단부터)

**의사결정의 시작은 정확한 진단** — 모든 dome-shaped radiolucency를 AP/MRC로 가정하면 misdiagnosis 위험.

### CBCT 평가 standard
[[sinus-lift/pseudocyst/yeung-2022-cbct-maxillary-sinus-pathology-review]] (HKU OMFS, Diagnostics 2022)는 CBCT를 통한 sinus pathology 평가 standard 정리:

| 항목 | 정상 | 병리 시사 |
|---|---|---|
| **Sinus Mucosal Thickness (SMT)** | <2 mm | ≥2 mm |
| **Antral Pseudocyst (AP)** | 없음 | Dome-shaped, base on sinus floor |
| **MRC** | 없음 | 비슷한 dome — size 추적 필요 |
| **Mucocele** | 없음 | Sinus 전체 침범 가능 (ENT 협진 강제) |
| **Septa** | 47-58% (population 의존) | 술식 위험 (Yang 2024 OR 4.03) |
| **Ostium patency** | 개방 | 폐쇄 → sinusitis 위험 |

### 감별 framework
[[sinus-lift/pseudocyst/nosaka-2024-sinus-elevation-radiopaque-lesions-review]] (Showa Univ Japan, JCM 2024)는 "Faintly radiopaque lesion"이 단순 AP가 아닐 수 있음 경계:

| 가능성 | 영상 특징 | 추가 검사 | Sinus elevation |
|---|---|---|---|
| Antral Pseudocyst (AP) | Dome, base on floor | CBCT 충분 | 가능 |
| MRC | 비슷한 dome | CBCT, 크기 측정 | 처치 후 가능 |
| Odontogenic cyst | 치아 관련, 잘 정의된 경계 | CBCT, 발치 후 enucleation | 처치 후 |
| Antrolith | 매우 hyperdense | CBCT | 제거 후 |
| Mucocele | 부비동 전체 침범 | CBCT + MRI | ENT 협진 |
| Tumor (희귀) | 침습성 경계 | MRI + 조직검사 | **금기** |

### 병리·해부 분류 (한국 임상)
[[sinus-lift/pseudocyst/jkaoms-48-2-101]] (원광대 OMFS, JKAOMS 2022)이 한국어 reference로 정리:

| Type | 상피층 | 형성 메커니즘 |
|---|---|---|
| **Antral Pseudocyst (AP)** | 없음 (가성낭종) | 염증 삼출액의 골막하 축적 |
| **Mucous Retention Cyst (MRC)** | 있음 (진성낭종) | 점액선 관 폐쇄 |
| **Mucocele** | 있음 (진성낭종) | 드물고 치명적 — 부비동 전체 침범 가능 |

**Synthesis [claude해석]:** AP와 MRC의 영상 구분은 어렵고 (둘 다 dome-shaped) 실제 구분은 조직학적 — 임상 결정에서는 size·증상·동반 sinusitis 신호가 더 중요한 결정 인자. ENT 협진 trigger는 (a) lesion >20mm + ostium 근처, (b) 부비동 전체 침범 의심, (c) 침습성 또는 일측성 만성 sinusitis 동반. [합의수준]

## Cluster 1 — 위험 자체 평가 (Step 1: 낭종이 있으면 implant 위험한가?)

[[sinus-lift/pseudocyst/anitua-2021-mucous-cysts-pseudocysts-implants-sr]] (Med Oral 2021, SR) — **결정 트리의 1단계 SR-level anchor**:

- 결론: **임상적으로 유의한 위험 증가 미입증**
- Implant survival은 일반적으로 높게 유지
- Routine implant 식립에 **절대적 금기 아님**
- 단, stratified approach 필요 (subtype·size별)

[근거강함] 본 SR이 분야 1차 question에 답함 → 처치 옵션 선택 (Cluster 2) 문제로 좁혀짐.

**Synthesis [claude해석]:** Anitua 2021 SR이 부정적 결론을 냈다면 적극 처치는 과잉치료가 될 뻔했다. SR-level 신호가 "위험 자체는 크지 않다"이므로 의사결정은 "처치 방법 선택"으로 압축된다. 단 SR이 RCT 부족·heterogeneity 높음 — "위험 없음"보다 "유의한 위험 미확인"이 더 정확한 표현. 후속 cohort (Fu 2022·2025, Yu 2023)가 size·subtype별 stratification 보강.

## Cluster 2 — 처치 옵션 결정 (Step 2: 어떻게 처치할 것인가)

핵심 cluster — 결정 트리의 본체. **4 옵션** (제거·흡인·미처치·전체 enucleation)을 **subtype·size 기반**으로 차등 적용.

### Subtype 분류 framework
[[sinus-lift/pseudocyst/lee-2024-pseudocyst-mrc-sinus-augmentation-cases]] (서울대 OMFS, JCFS 2024)이 결정 트리를 case로 시각화:

| Type | Pathology | Size | 권장 술식 |
|---|---|---|---|
| **Type 1** | Antral Pseudocyst (PsC) | — | Spontaneous drainage → augmentation |
| **Type 2** | Mucous Retention Cyst (MRC) | <20 mm | Aspiration → augmentation |
| **Type 3** | MRC | >20 mm | Enucleation → 회복 → augmentation |

[claude해석] 20mm cut-off는 본 case series에서 가설적 — Fu 2025 cohort (n=107)는 ≤20mm 전체를 한 군으로 다뤄 cut-off 직접 validate 못함. 임상 적용 시 size + ostium 근접도 + 동반 증상을 종합 판단.

### 4 옵션의 정량 비교

#### A. Timing (제거 후 즉시 vs 지연)
[[sinus-lift/pseudocyst/yu-2023-immediate-delayed-sinus-pseudocyst-rct]] (Peking Univ, RCT n=31/33 sinuses, CIDRR 2023):
- One-stage (낭종 제거 + 즉시 augmentation) vs Two-stage (3개월 지연)
- **신생골·implant 생존·합병증·VAS 통증 모두 유의차 없음**
- **One-stage 접근 타당** — 3개월 대기로 인한 추가 회복기간이 결과 개선 안 함

[근거강함] 분야 RCT는 본 1편 — "제거 후 언제?" 결정에 직접 anchor.

#### B. 3-arm 비교 (제거 vs 흡인 vs 미처치)
[[sinus-lift/pseudocyst/fu-2025-antral-pseudocyst-management-lsfe]] (Zhongshan Hospital Fudan, retrospective n=107, BMC Oral Health 2025) — 소형 (≤20mm) AP 한정:

| 옵션 | 6개월 골이식재 변화 | Implant 생존 | Sinusitis |
|---|---|---|---|
| Removal | -14.11% [-17.42, -11.05] | 100% | 0% |
| Aspiration | -16.55% [-18.71, -11.83] | 100% | 0% |
| Untreated | -18.31% [-19.72, -15.24] | 100% | 0% |
| P value | <.001 | NS | NS |

→ **모든 옵션 implant 생존 100%**, **미처치 시 골이식재 -4%p 더 흡수**

[claude해석] 미처치(observation)도 implant 자체 안전. 단 grafted bone 보존성 측면에서 제거 > 흡인 > 미처치 순. 환자가 침습 거부하거나 매우 작은 AP면 미처치 정당화 가능, 단 graft volume 손실 감수.

#### C. Aspiration (중간 옵션)
[[sinus-lift/pseudocyst/testori-2024-mucosal-cyst-aspiration-sinus-cohort]] (Testori·Del Fabbro·Wang 그룹, CIDRR 2024, cohort n=15/18 sinuses):
- Needle aspiration + 즉시 sinus elevation
- 합병증 minimal, implant placement 가능
- **중간 옵션의 prospective 표준화**

[합의수준] Testori 그룹 (Galeazzi·ARDEC·Michigan) 권위로 임상 신뢰도 높음.

#### D. Enucleation (대형 MRC)
[[sinus-lift/pseudocyst/jadach-2024-croco-eye-mrc-sinus-lift-cohort]] (Wrocław, JCM 2024, retrospective cohort):
- "Croco Eye" technique — 측방창 통해 대형 MRC enucleation + 동시 augmentation
- One-stage 가능 — Yu 2023 RCT의 timing 결론과 일관

#### E. 2-arm 비교 (제거 vs 잔존)
[[sinus-lift/pseudocyst/fu-2022-pseudocyst-removal-retainment-msfa-cohort]] (Zhejiang Univ, retrospective n=26/38 implants, COIR 2022) — 양쪽 옵션 모두 immediate implant 가능 (조건부)

### Korean 임상 case anchor (참조용)
- [[sinus-lift/pseudocyst/jkaoms-48-2-101]] — 원광대 OMFS 6 case, 분류·해부 review
- [[sinus-lift/pseudocyst/jkda-2022-60-4-002]] — 전북대 치주과 윤정호 그룹 case (치주과 perspective)

### Transcrestal 분기 (다른 접근법)
[[sinus-lift/pseudocyst/wang-2023-antral-pseudocyst-drift-osteotome-case]] (Sichuan Univ, JCM 2023):
- OSFE (osteotome) + 동시 implant 후 AP drift 현상
- Transcrestal 접근의 드문 datapoint
- Lateral 중심 결정 트리와 별도 분기

[claude해석] Transcrestal 접근은 RBH 충분 (>5mm) + 작은 AP에서 가능 — [[sinus-lift/transcrestal/sirinirund-2025-minimally-invasive-sinus-augmentation-review]] (VSA threshold ≥5mm) 와 결합 결정.

## Cluster 3 — 장기 Trajectory (Step 3: 처치 후 어떻게 변화하는가)

처치 결정 후 환자에게 "10년 후 어떻게 될까?"에 답하기 위한 case-based trajectory anchor.

### Longitudinal datapoints (점·기간별)

| Case | 추적 | 처치 | 결과 |
|---|---|---|---|
| [[sinus-lift/pseudocyst/berberi-2023-mucous-retention-cyst-sinus-lift]] (Beirut) | **1년** | MRC + lateral sinus lift + 즉시 implant | 합병증 없이 정착, implant 생존 |
| [[sinus-lift/pseudocyst/silva-2025-sinus-lift-mrc-drainage-6year]] (USP-ARDEC) | **6년 + 조직학** | MRC drainage + lateral SFE → guided implant | Implant·grafted bone·sinus health 모두 안정, vital bone 형성 확인 |
| [[sinus-lift/pseudocyst/grun-2023-antral-pseudocyst-15year-sinus-implant-case]] (Austria) | **15년** | AP **잔존(untreated)** + sinus augmentation + implant | 15년 implant 정착·기능, AP radiographic 동적 변화 (size 증·감), major 합병증 없음 |

**Synthesis [claude해석]:**
- 1년 (Berberi) → 단기 안정성 확인
- 6년 (Silva) → 중기 trajectory + 조직학 (vital bone)
- 15년 (Grün) → **untreated 옵션의 장기 anchor** — 처치 안 해도 implant 자체 안전 가능
- 단 모두 case report (n=1) — 대규모 cohort의 5년+ 데이터는 분야 gap

→ 환자 설명에 직접 인용 가능 — "유사 case가 6년·15년까지 정상 추적된 보고 있음"

## 통합 결정 트리 (Step-by-Step Workflow)

```
[Step 0] CBCT 평가
  ├─ SMT <2mm, lesion 없음 → 일반 sinus lift 진행
  ├─ Dome-shaped lesion 발견 → AP/MRC 의심
  └─ Atypical (침습성·전체 침범) → ENT 협진 + MRI/조직검사

[Step 1] AP/MRC 확인 시 위험 평가
  ├─ Anitua 2021 SR → 임상적 유의 위험 미입증
  └─ → Step 2로 진입 (처치 옵션 선택)

[Step 2] Subtype + Size 기반 처치 결정
  ├─ Type 1 (PsC, 무증상 작음) 
  │   → Option A: Untreated (graft -18%, implant 생존 100%) — 환자 부담 최소
  │   → Option B: Aspiration (graft -16%) — 중간 옵션
  │   → Option C: Removal/spontaneous drainage (graft -14%) — graft 보존성 최우선 시
  │
  ├─ Type 2 (MRC <20mm)
  │   → Aspiration (Testori 2024 standard) → 동시 sinus lift
  │   → 또는 Removal + 동시 sinus lift (Yu 2023 RCT — 즉시 OK)
  │
  ├─ Type 3 (MRC >20mm) 또는 시야 가림·천공 위험
  │   → Enucleation (Croco Eye technique, Jadach 2024)
  │   → 동시 augmentation (Yu 2023 RCT — 3개월 대기 불필요)
  │
  └─ Transcrestal 분기 (RBH ≥5mm + 작은 AP)
      → OSFE + 동시 implant 가능 (Wang 2023 case)
      → 단 AP drift 가능성 인지

[Step 3] 장기 trajectory 모니터링
  ├─ 1년 (Berberi) → 단기 안정성
  ├─ 6년 (Silva) → 중기 + 조직학 ✓
  └─ 15년 (Grün, untreated) → 장기 anchor
```

## 한국 임상 적용 framework (Practical Workflow)

### 외래 1회차 (진단·계획)
1. CBCT 표준 평가:
   - SMT 측정
   - Lesion (있다면) size·position·base
   - Septa, ostium patency
   - 동반 sinusitis 신호 (mucosal thickening 광범위)
2. 감별진단 (Nosaka 2024 framework):
   - 단순 AP/MRC vs odontogenic cyst vs mucocele vs tumor
3. ENT 협진 trigger 평가:
   - Lesion >20mm + ostium 근처
   - 부비동 전체 침범 의심
   - 침습성·일측성 만성 sinusitis 동반

### 외래 2회차 (수술 계획·환자 설명)
| Subtype | 환자 설명 요지 | 동의서 항목 |
|---|---|---|
| Type 1 PsC | "낭종은 위험인자 아님, 미처치도 옵션" | Untreated risk: graft 일부 흡수 |
| Type 2 MRC <20mm | "흡인 또는 제거 후 즉시 골이식·임플란트" | Aspiration 합병증·재발 가능성 |
| Type 3 MRC >20mm | "낭종 제거 후 즉시 또는 3개월 후 골이식" | Enucleation·membrane 천공 risk |

### 외래 3회차+ (수술·추적)
- 6개월: CBCT graft 평가, implant 식립
- 12개월: prosthetic loading, sinus health 평가
- 매년: 정기 CBCT (3-5년), 이후 2년마다

### Hygienist 매뉴얼 체크리스트
1. 수술 전 환자 코·부비동 증상 history (코막힘·후비루·악취)
2. 흡연 history (sinus 합병증 risk factor)
3. 수술 후 1주·1개월 sinusitis 증상 모니터링
4. CBCT 정기 follow-up 예약 (1년·3년·5년)

## Strengths and Gaps

### 강점
- [근거강함] Anitua 2021 SR — 결정 1단계 anchor
- [근거강함] Yu 2023 RCT (timing) + Fu 2025 cohort (n=107, 3-arm) — strong evidence for option selection
- [합의수준] Subtype framework (Lee 2024 type 1/2/3) + Croco Eye (Jadach 2024) — surgical playbook
- [근거강함] Grün 2023 15년 case — 분야 최장 untreated trajectory anchor

### 약점·gap
- [미검증] 20mm cut-off (Lee 2024 type 2/3 경계)는 가설적 — RCT validation 없음
- [미검증] 대형 (>20mm) MRC enucleation의 large cohort 부족 (Jadach 2024 외 소수)
- [claude해석] Transcrestal 접근에서 AP/MRC 처치 RCT 없음 (Wang 2023 case 1편)
- [미검증] 한국 cohort 분포·임상 결과 데이터 (jkaoms·jkda는 소수 case)
- [claude해석] 장기 (≥10년) cohort 부족 — Grün 15yr case 외 datapoint 없음

### 향후 보강 필요
- 20mm cut-off validation RCT
- Transcrestal + AP/MRC RCT
- Korean cohort 다기관 prospective study
- 10년+ 대규모 cohort
- ENT 협진 trigger 정량화 (objective criteria)

## Bottom Line (한 줄 결론)

[근거강함] **상악동 점액낭종/위낭종 자체는 implant 절대 금기 아님** — Anitua 2021 SR 신호. 처치 결정은 subtype·size·증상 기반 stratification.

[합의수준] Subtype framework (Type 1 PsC / Type 2 MRC <20mm / Type 3 MRC >20mm) + 차등 술식 (drainage·aspiration·enucleation). One-stage (즉시 augmentation)이 two-stage와 동등 — Yu 2023 RCT.

[claude해석] 한국 임상 default: **Type 1 → Untreated/Aspiration 옵션 환자에게 제시, Type 2 → Aspiration + 즉시 augmentation, Type 3 → Enucleation + 즉시 augmentation**. 장기 trajectory는 15년 untreated case (Grün 2023)까지 안전 datapoint — 환자 설명에 직접 인용 가능. ENT 협진 trigger (>20mm + ostium 근접, 부비동 전체 침범, 침습성)는 명확히 설정해두고 의심 시 즉시 의뢰.

[claude해석] 20mm cut-off는 본 framework의 가설적 기준 — RCT validation 없음. 실제 임상에서는 size + ostium 근접도 + 동반 증상 + 술자 시야 확보 가능성을 종합 판단.
