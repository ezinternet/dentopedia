## 한줄요약

구강내과 7축 — OPMD 일반 · OLK 악성전환 monitoring · OLP 진단·관리 · BMS · RAS · OMS · 수포·궤양성 면역매개 점막질환(HSV·EM·GVHD·PFAPA). WHO consensus + Cochrane SR+MA + Delphi 합의 spine.

## Summary

본 페이지는 wiki/oral-medicine/ paper의 7-axis spine. 구강내과는 narrative review 위주이나 OLK 악성전환·OLP 치료(Lodi 2020 Cochrane)는 SR+MA, OPMD 명칭은 WHO consensus가 spine. 2026-06 herpes/EM/GVHD/PFAPA 클러스터(11편) 추가로 수포·궤양성 면역매개 점막질환 축(축 7) 신설.

핵심 명제 5개:
1. **OLK (Leukoplakia) 5년 악성전환률 3.5% (pooled). 비균질형 OR 4.1 — 균질형 단순 monitoring, 비균질형 적극 biopsy** — Aguirre-Urizar 2021 SR+MA. [근거강함]
2. **OLP (Oral Lichen Planus) 진정한 악성전환률 0.5-1.4% (고품질 연구만). 과거 5% 보고는 진단 오류 가능성** — Gonzalez-Moles 2021 SR, Kaur 2022 비판. [합의수준]
3. **OPMD 명칭·분류는 WHO 2020 consensus 표준화** — Warnakulasuriya 2021 WHO consensus. [근거강함]
4. **BMS 1차 치료 — clonazepam 국소 도포 + CBT + 저용량 TCA. ICD-11에서 BMD로 재명명 합의** — Canfora 2026 review, Chmieliauskaite 2020 Delphi. [합의수준]
5. **단측 상악동 혼탁의 30-72%가 치성 원인 (OMS). 치과+ENT 다학제 + ESS 병행 표준** — Lin 2024 expert consensus, Psillas 2020 review. [근거강함]

### 축 2 — OLK (Leukoplakia)

| Spine paper | Evidence | Key finding |
|---|---|---|
| [[oral-medicine/aguirre-urizar-2021-leukoplakia-malignant-transformation-sr-ma]] | sr+ma | 5y 악성전환 pooled 3.5%. 비균질형 OR 4.1 |

**임상 ladder (OLK)**:
1. **균질형 (homogeneous) + 작은 병소 (< 2 cm) + 위험인자 (흡연·음주) 통제 가능** → 위험인자 제거 + 3-6mo monitoring + 사진 기록.
2. **비균질형 (non-homogeneous: speckled, nodular, verrucous)** → biopsy 적극. OR 4.1 위험.
3. **고위험 부위 (혀 측면·구강저)** + 어떤 형태든 → biopsy 우선.
4. **이형성 (dysplasia) 진단** → 외과 절제 + 정기 follow-up.

| [[oral-medicine/kaur-2022-oral-lichen-planus-malignant-disorder-appraisal]] | narrative-review | OLP를 OPMD로 분류 비판 — 진단 오류 가능성 |

| [[oral-medicine/chiang-2021-lichen-planus-malignant-transformation-review]] | narrative-review | 위험인자·6mo 임상+조직학 평가 권고 |
| [[oral-medicine/lodi-2020-oral-lichen-planus-corticosteroids-cochrane]] | sr+ma | 국소 스테로이드 vs 위약 통증해소 RR 1.91 (낮은 근거); 타크롤리무스>클로베타솔 가능 (매우 낮은 근거) |
| [[oral-medicine/sandhu-2022-oral-lichen-planus-treatment-sr]] | sr | 효능+비용 비교 — 스테로이드가 효능-비용 기준, 타크롤리무스 효과적이나 고가 |

**임상 ladder (OLP)**:
1. **Reticular (망상) — 무증상** → 위험인자 (흡연·음주) 통제 + 6mo 임상 + 사진 기록. 약물 불필요.
2. **Erosive·atrophic — 증상 있음** → 국소 스테로이드 (clobetasol 0.05% ointment BID 2-4w → tapering). 근거: Lodi 2020 Cochrane — 위약 대비 통증해소 우월 가능하나 근거수준 낮음. [근거강함(SR+MA)이나 effect certainty 낮음]
2-b. **스테로이드 불응·부작용** → 국소 tacrolimus 0.1%. Lodi 2020에서 통증해소 우월 가능성(매우 낮은 근거). 비용은 Sandhu 2022 — tacrolimus 고가.
3. **OLP-like 병변** → biopsy로 확진 (Van der Meij 기준).
4. **악성전환 monitoring** — 진정한 위험 0.5-1.4%. 과대 위험 인식 회피. 단 erosive·atrophic에서 위험 약간↑.
5. **약물 유발 (lichenoid drug reaction)** — 의심 약물 stop + 내과 협진.

### 축 4 — BMS (Burning Mouth Syndrome / Disorder)

| Spine paper | Evidence | Key finding |
|---|---|---|
| [[oral-medicine/canfora-2026-burning-mouth-syndrome-management-review]] | narrative-review | 1차 — clonazepam 국소, CBT, 저용량 TCA. 다학제 |
| [[oral-medicine/canfora-2025-burning-mouth-syndrome-pathogenesis-review]] | narrative-review | 병인 — 신경병증·심리·호르몬 통합 모델. ICD-11 진단 알고리즘 |
| [[oral-medicine/chmieliauskaite-2020-burning-mouth-syndrome-icd11-rename-delphi]] | consensus (Delphi) | BMS → BMD 재명명. ICD-11 기준 개선 |
| [[oral-medicine/gupta-2016-burning-mouth-syndrome-review]] | narrative-review | 폐경 후 여성 0.7-4.6% 유병률 |

**임상 ladder (BMS)**:
1. **진단 — 2차 원인 제거** → 빈혈·당뇨·갑상선·SS·Candida·치과 보철 적합도 + 약물 부작용 검토.
2. **1차 약물** → Clonazepam 1 mg 국소 (입에 머금기) TID 2-3w.
3. **CBT** — 심리·정서 요소 큰 환자.
4. **저용량 TCA** — Amitriptyline 10-25 mg HS 또는 nortriptyline. 만성 신경병증 통증 처치 원칙.
5. **호르몬 보충** — 폐경 후 여성 + 부인과 협진.

### 축 5 — RAS (Recurrent Aphthous Stomatitis)

| Spine paper | Evidence | Key finding |
|---|---|---|
| [[oral-medicine/manfredini-2021-recurrent-aphthous-stomatitis-treatment]] | narrative-review | 국소 스테로이드 1차. 콜히친·탈리도마이드 중증 |
| [[oral-medicine/mills-2017-aphthous-stomatitis-oral-medicine-review]] | narrative-review | 병인 — 면역·유전·영양·스트레스. 단계적 접근 |
| [[oral-medicine/belenguer-guallar-2014-recurrent-aphthous-stomatitis-review]] | narrative-review | 국소 스테로이드·항균제·NSAIDs·LLLT 비교 |
| [[oral-medicine/gasmi-2021-oral-aphthous-pathophysiology-treatment]] | narrative-review | 병태생리·세 형태·감별; 국소 스테로이드 1차·진행 시 전신제 |
| [[oral-medicine/manthiram-2023-pfapa-syndrome-pathogenesis-review]] | narrative-review | PFAPA — 재발성 아프타의 전신 autoinflammatory 원인. 편도절제술 효과 |

**임상 ladder (RAS)**:
1. **Minor RAS (< 1 cm, ≤ 5 lesions, 1-2w 자연 치유)** → 국소 스테로이드 (triamcinolone in Orabase 0.1% BID) + 위안.
2. **Major RAS (> 1 cm, 3-6w 지속)** → 국소 스테로이드 + clobetasol gel.
3. **Herpetiform** — 다수 작은 ulcer cluster → 국소 스테로이드 + 영양 검사.
4. **중증·재발 빈번 (≥ 6/y)** → 콜히친 0.5-1 mg/d 또는 thalidomide (Behçet 의심 시) — 내과·구강내과 협진.
5. **영양 검사** — 철·B12·엽산·아연 결핍 확인. 결핍 있으면 교정.
6. **소아 + 규칙적 주기열 동반 (편도염·경부림프절염)** → PFAPA 의심. 소아과·류마티스 협진. 편도절제술이 효과적일 수 있음 (Manthiram 2023). 베체트병 감별.

### 축 6 — OMS (Odontogenic Maxillary Sinusitis)

| Spine paper | Evidence | Key finding |
|---|---|---|
| [[oral-medicine/lin-2024-odontogenic-maxillary-sinusitis-expert-consensus]] | consensus | 단측 상악동 혼탁 45-72%가 치성. 치과+ENT 다학제 |
| [[oral-medicine/psillas-2020-odontogenic-maxillary-sinusitis-comprehensive-review]] | narrative-review | 단측 30% 치성. 발치·치성감염·OAF가 흔한 원인 |

**임상 ladder (OMS)**:
1. **단측 상악 후방 sinusitis** → 치과 origin 우선 의심. 의심 치아 (특히 1번·2번·3번 대구치) 평가.
2. **CT 또는 CBCT** — 단측 혼탁 확인 + 치성 origin 단서 (apex 침범, OAF, 잔존 root tip, 임플란트 등).
3. **치과 처치** — 원인 치아 endo·extraction·OAF 폐쇄·임플란트 적합도 평가.
4. **ENT 협진** — Functional endoscopic sinus surgery (FESS) 필요한 경우.
5. **항생제** — Amoxicillin-clavulanate 7-10d. 만성·재발 시 culture-guided.

### 축 7 — 수포·궤양성 & 면역매개 점막질환 (Vesiculo-ulcerative & immune-mediated)

감염성(HSV)·면역매개(EM·GVHD)·autoinflammatory(PFAPA) 점막질환을 묶는 축. RAS(축 5)와 임상적으로 감별 대상이며, 다수가 HSV를 공통 trigger로 공유한다.

| Spine paper | Evidence | Key finding |
|---|---|---|
| [[oral-medicine/coppola-2023-herpetic-gingivostomatitis-sr]] | sr | 1차 헤르페스 치은구내염(PHGS, 소아) — 조기 아시클로버 약한 근거만. ~72h 진단지연이 효능 저해 |
| [[oral-medicine/mancini-2025-oral-herpes-treatment-sr]] | sr | 재발성/구순 HSV — 국소제는 병변기간만 단축(재발 미감소), 전신 항바이러스제는 재발 예방 |
| [[oral-medicine/lv-2025-anti-hsv1-agents-update]] | narrative-review | 항-HSV-1 약리 — 뉴클레오시드 유사체 1차, 내성 시 helicase-primase 억제제 |
| [[oral-medicine/kechichian-2024-erythema-multiforme-review]] | narrative-review | 다형홍반(EM) — HSV/마이코플라스마 유발. SJS/TEN 감별. 재발형은 억제 항바이러스 1차 |
| [[oral-medicine/dean-2022-oral-chronic-gvhd-review]] | narrative-review | 구강 만성 GVHD(동종 HCT 30-50%) — 태선양·건조·경화. NIH 2014 기준. 국소→전신 면역억제 |
| [[oral-medicine/manthiram-2023-pfapa-syndrome-pathogenesis-review]] | narrative-review | PFAPA autoinflammatory 주기열 — 재발성 아프타+발열. 편도절제술 효과 |

**임상 ladder (수포·궤양성)**:
1. **다발 수포·전신증상(발열) 소아 + 급성 발병** → 1차 PHGS 의심. 조기(<72h) 아시클로버 고려 + 수분·진통 지지요법. 탈수 시 입원 (Coppola 2023).
2. **재발성 구순포진(herpes labialis)** → 조기 국소 아시클로버/도코사놀(병변기간 단축). 빈번 재발(≥6/y) 시 전신 억제요법 valaciclovir (Mancini 2025).
3. **표적병변(target lesion) + 점막 침범** → EM 의심. Mycoplasma 우선 배제(항생제 필요), 재발형은 HSV 유발 → 억제 항바이러스 (Kechichian 2024). 광범위 점막탈락이면 SJS/TEN 응급 감별.
4. **동종 조혈모세포이식 병력 + 태선양 점막염·구강건조·개구장애** → 구강 cGVHD. NIH 2014 staging, 국소 스테로이드 rinse→전신 면역억제. 악성전환 monitoring (Dean 2022).
5. **항바이러스 내성 의심(면역저하)** → helicase-primase 억제제(pritelivir/amenamevir) 고려 (Lv 2025).
