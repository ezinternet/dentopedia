## 한줄요약
OPG 위험 징후 → CBCT로 IAC 위치 확인 → 설측 위치(OR 4.96)·고위험 환자면 관상절제술 선택 — IAN 손상 0.59%로 완전 발치 대비 유의하게 낮춤.

---

## Summary

하악 제3대구치 (lower third molar, LM3) 발치는 구강외과 수술 중 가장 빈번하게 시행되지만, 0.35–8.4%에서 하치조신경 (Inferior Alveolar Nerve, IAN) 손상이 발생한다. 이 overview는 5편의 근거를 통합하여 **발치 전 IAN 위험 평가 → 영상 단계적 선택 → 술식 결정**의 임상 흐름을 정리한다.

---

## 1. 배경: IAN 손상 유병률과 예후

| 지표 | 수치 | 출처 |
|---|---|---|
| IAN 감각이상 발생률 | **0.35–8.4%** | Sarikov 2014 |
| 영구적 손상 | 드묾 (< 1%) | Sarikov 2014 |
| 4–8주 내 자연 회복 | **96%** | Sarikov 2014 |
| 설측 IAC 위치 시 IAN 손상 OR | **4.96** (95% CI 2.11–11.62) | Li 2022 |

> 범위가 0.35–8.4%로 넓은 이유: 술자 경험, 매복 양상, 환자 연령, 영상 평가 방법이 모두 다르기 때문.

---

## 2. 위험인자 — 발치 전 확인 체크리스트

### 2-1. 환자 요인
| 위험인자 | 근거 |
|---|---|
| 나이 **>24세** | 치근 완성 → 신경과 더 가까워짐 (Sarikov 2014) |
| 수평 매복 (horizontal impaction) | 가장 위험한 매복 각도 (Sarikov 2014) |
| 수련의 시술 | 경험 많은 술자 대비 위험↑ (Sarikov 2014) |

### 2-2. 영상 요인 (OPG 위험 징후 — Rood & Shehab signs)
아래 중 **1가지라도** 있으면 CBCT 적응증:

| OPG 소견 | 의미 |
|---|---|
| 신경관 협착 (narrowing of canal) | 치근이 IAC를 압박 |
| 신경관 편향 (deflection) | 치근이 IAC를 밀어냄 |
| 신경관 피질골 소실 (loss of lamina dura) | 직접 접촉 가능성 ↑ |
| 치근 굴절·암화 (root darkening/grooving) | 신경관 내부 통과 가능성 |
| 치근 끝 협착 | 신경 포획 가능성 |

→ 각 징후가 독립적으로 **IAN 손상 위험 약 30%** 예측 (Sarikov 2014)

---

## 3. CBCT — IAC 위치 분류와 위험도

OPG 위험 징후 확인 후 CBCT 촬영. 관상면(coronal view)에서 IAC 위치를 분류:

```
협측(Buccal)   <──── 치근 ────>   설측(Lingual)
    낮은 위험                        높은 위험
                                    OR 4.96
```

| IAC 위치 | IAN 손상 위험 | 임상 해석 |
|---|---|---|
| **협측 (Buccal)** | 낮음 | 기준군 — 완전 발치 고려 가능 |
| **근심–원심 사이 (Inter-radicular)** | 중간 | 신중한 접근 |
| **설측 (Lingual)** | **OR 4.96×** 높음 | 치근 발치 시 IAC 포획 위험; 관상절제술 강적응증 |
| **하방 (Inferior)** | 상대적으로 낮음 | 직접 접촉 적음 |

*Li et al. SR+MA 11편, 고이분산성(I²=65%) 인정 — OR은 방향성 지표로 활용* (Li 2022)

## 4. 발치 적응증 결정 — NICE·AAOMS 가이드라인

| 적응증 분류 | 상태 | 권고 |
|---|---|---|
| **절대적응증** | 반복 관주위염 (recurrent pericoronitis) | 발치 |
| | 농양·봉와직염 | 발치 |
| | 낭종·종양 연관 | 발치 |
| | 인접 제2대구치 치근 흡수 | 발치 |
| | 교정 필요 공간 | 발치 |
| **상대적응증** | 25세 미만, 매복 패턴상 문제 예측 | 예방적 발치 고려 |
| **경과 관찰** | 무증상 완전 매복, 병리 없음 | 정기 추적 |

*(D'Amico 2025 — NICE / AAOMS)*

## 5. 술식 선택 플로우차트

```
제3대구치 내원
       │
       ▼
OPG 판독
       │
       ├─ Rood & Shehab 징후 없음 ──────────────────────┐
       │                                                │
       └─ Rood & Shehab 징후 1개 이상 ──▶ CBCT 촬영    │
                                              │         │
                               ┌─────────────┼──────┐  │
                          협측/하방         설측/근심  │  │
                               │               │     │  │
                               ▼               ▼     │  │
                      위험도 낮음         위험도 높음   │  │
                               │        (OR ≈ 5×)   │  │
                               │               │     │  │
    ┌──────────────────────────┼───────────────┘     │  │
    │                          │                     │  │
    ▼                          ▼                     │  │
완전 발치 고려            관상절제술 (Coronectomy)  ◀──┘  │
    │                    고려 (강적응증)                  │
    │                                                    │
    └────────────────────────────────────────────────────┘
                (추가 위험인자 없을 때: 나이 <24, 비수평 매복)
```

**단, 관상절제술 금기:**
- 치근 감염(근관치료 필요)
- 치관 아래 치근이 이미 신경관과 완전 융합
- 수직·원심 매복으로 치관 제거 자체가 불가한 경우

---

## 6. 관상절제술 (Coronectomy) — 근거와 합병증 프로파일

> 치관만 제거, 치근은 잔존 → IAN 직접 조작 회피

### 6-1. 완전 발치 대비 우위
| 결과 | 비교 | 출처 |
|---|---|---|
| IAN 감각 손실 | **유의하게 감소** | Cervera-Espert 2016 SR+MA |
| 건성발치창 (dry socket) | **유의하게 감소** | Cervera-Espert 2016 SR+MA |
| 동통 | 차이 없음 | Cervera-Espert 2016 SR+MA |
| 감염 | 차이 없음 | Cervera-Espert 2016 SR+MA |

### 6-2. 관상절제술 합병증 프로파일 (2,176례)
| 합병증 | 발생률 | 해석 |
|---|---|---|
| IAN 손상 | **0.59%** | 완전 발치(0.35–8.4%) 하한보다 낮음 |
| 설측 신경(LN) 손상 | 0.22% | 드묾 |
| 감염 | 3.95% | 관리 가능 |
| 건성발치창 | 1.12% | 낮음 |
| 치근 재발치 필요 | 5.28% | 증상 발생 시 재수술 |
| 재개입 전체 | 1.13% | 낮음 |
| 통증 (가장 흔한 호소) | 22.04% | 통상 발치와 유사 |

*(Póvoa 2021 SR, 16편)*

### 6-3. 치근 이동 (Root Migration)
- 잔존 치근은 **평균 2년 내 약 2mm 치관 방향으로 이동** (Cervera-Espert 2016)
- 대부분 자연 경과로 허용 범위 내; 재발치가 필요한 경우는 소수
- 정기 OPG 추적으로 감시

---

## 7. 임상 요약 — 1페이지 핵심

| 단계 | 핵심 행동 |
|---|---|
| **Step 1** OPG | Rood & Shehab 징후 ≥1개 → CBCT 처방 |
| **Step 2** CBCT | IAC 위치 분류: 설측 → 고위험 플래그 |
| **Step 3** 위험 적층 | 설측 IAC + 나이 >24 + 수평 매복 → 관상절제술 강적응증 |
| **Step 4** 술식 결정 | 고위험: 관상절제술 / 저위험: 완전 발치 |
| **Step 5** 관상절제술 후 | 2년 OPG 추적; 치근 이동 확인 |
