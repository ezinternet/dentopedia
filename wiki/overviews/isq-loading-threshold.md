## 한국어 핵심요약

> [!summary] 한국어 핵심요약
> - 핵심 명제: 임플란트 안정성 지수 (Implant Stability Quotient, ISQ) 부하 결정의 견고한 임계값은 **<70 위험·≥70 안전** (Andersson 2019, n=745, 5년 누적생존율 (CSR) 97.3%). ISQ 언급 27편 횡단 합성.
> - ISQ는 단독 결정 지표가 아니다: 골질·삽입토크 (Insertion Torque, IT)·표면처리·시점 5요소와 함께 해석. ISQ–IT 상관은 r=0.44, I²>90%로 중등도이며 비교 불가 (Tisci 2026 SR+MA) — 두 지표 모두 측정 권장.
> - 원형 임계값 (Sennerby 2008, Oh 2008): ISQ <60 지연부하, 60–70 조기부하+모니터링, >70 즉시/조기부하. 핵심은 단일 70 컷오프가 아니라 **60–70 회색지대의 존재**.
> - 즉시부하 결정: ISQ ≥70 + IT ≥35 Ncm 양조건 충족 시 안전 (Wittneben 2023 SR). IT >40 Ncm 단독으로도 6–8주 조기부하 가능 (Andreatta 2020).
> - 골질별 보정 (Lekholm-Zarb): Type I–III는 표준 임계값 (≥70), **Type IV (상악 구치부)는 ISQ 58–65이 정상 분포**이므로 70 미달이라고 자동 지연 결정 아님. 실패 위험은 Type I 대비 1.92배 (Rosa 2024).
> - Type IV 보정 수단: 친수성/생체활성 표면이 안정성 딥 (stability dip) 제거 (Canullo 2024), 골밀도화 (Osseodensification, OD)가 Type IV에서 ISQ 상승 최대 (Fontes-Pereira 2023).
> - 표면처리의 역할: "임계값 자체가 내려간다"가 아니라 "임계값 도달 속도가 빨라진다"가 정확. OD의 ISQ 우위는 일반식립에서 일시적 (3개월에 소멸)이나, 상악동거상 동시식립에서는 확실 (Starch-Jensen 2025 SR+MA).
> - 안정성 딥: 1차 (기계적) 안정성과 2차 (생물학적) 안정성 사이 2–4주 하락 구간. 즉시부하 Brånemark에서 ISQ 70.4→66.4(30일)→68.8(90일) 딥-회복이 전향적으로 문서화됨 (Balshi 2005, n=344). 연조직·구치부·여성에서 딥 더 크고 회복 더 느림. 전통 SLA·D3/D4 골에서 명확하나 생체활성 나노표면·친수성 표면에서는 거의 없음. 식립시 단일값보다 2–4주 후속 공진주파수분석 (RFA)이 더 신뢰성 높음.
> - 신선 발치와: ISQ가 인공적으로 낮음 (~57 vs 치유골 ~72, Bavetta 2019) → IT 값 (ITV) ≥35 Ncm을 1차 기준으로.
> - 측정 신뢰성: Osstell 급내상관계수 (ICC)=0.97로 단회 측정 충분, Osstell vs Osseo 100 호환 (r>0.95). SmartPeg 한 번 체결·협설 1회 측정이면 충분 (4방향 평균 불필요).
> - 6대 임상원칙: ① ISQ ≥70 표준 안전선, ② 즉시부하 ISQ ≥70+IT ≥35 Ncm, ③ 6–8주 조기부하 IT >40 Ncm, ④ D4 골 ISQ 58–65 정상, ⑤ 신선 발치와 ITV 우선, ⑥ 상악동 동시식립 OD 시 ISQ ~10 상승. ISQ 절대 단독 사용 금지.

## One-line Summary

Cross-category synthesis (27 ISQ-citing pages) building a confidence-graded matrix for the Implant Stability Quotient (ISQ) loading-decision threshold. Core thesis: the robust threshold is <70 risky / ≥70 safe (Andersson 2019, n=745, 5y CSR 97.3%), with immediate loading requiring ISQ ≥70 + insertion torque (IT) ≥35 Ncm (Wittneben 2023 SR); but ISQ is never a standalone decision metric — it must be read together with bone quality, IT, surface treatment, and timing (ISQ–IT correlation only r=0.44, I²>90%, Tisci 2026 SR+MA), Type IV bone normally runs ISQ 58–65 (do not auto-delay), fresh sockets read artificially low (use ITV), and surface treatment changes the speed of reaching threshold (stability dip) rather than the threshold itself.

## 한줄요약
ISQ 부하 결정의 견고한 임계값은 **<70 위험·≥70 안전**(Andersson 2019, n=745, 5년 CSR 97.3%)이고, ITV ≥35 Ncm 동반 시 즉시부하 가능(Wittneben 2023 SR); 단, ISQ는 단독 결정 지표가 아닌 골질·삽입토크·표면처리·시점과 함께 해석해야 함 (ISQ–IT 상관 r=0.44, I²>90%, Tisci 2026 SR+MA).

## 1. Foundation Thresholds — 어디서 왔는가

ISQ 부하 임계값의 원형은 Sennerby & Meredith (2008, narrative-review) 가 제시한 다음 구조에서 출발한다 [합의수준]:

| ISQ | 권고 | 근거 시점 |
|---|---|---|
| < 55–60 | 지연 부하 (Delayed loading) | 2008 narrative |
| 60–65 | 신중한 조기 부하 + 모니터링 | 2008 narrative |
| > 65–70 | 즉시/조기 부하 고려 가능 | 2008 narrative |

Oh & Chang (2008, in-vivo, 한국 임상 가이드) 가 동일 구조를 다음과 같이 단순화 [claude해석 — 근거 등급 in-vivo로 약함, 그러나 한국 임상 실무 표준]:

- **ISQ < 60**: 지연 부하 (4–6개월)
- **ISQ 60–70**: 조기 부하 + 8–12주 모니터링
- **ISQ > 70**: 즉시/조기 부하 가능

이 두 페이지가 모든 후속 임상연구의 기준선이 되었다. 핵심은 **70이라는 단일 컷오프가 아니라 60–70 회색지대의 존재**다.

**합성 결론** [근거강함]: **ISQ ≥70 at placement = 표준 부하 안전선, ISQ <70 = 위험 신호로 지연·재평가**. 단, 이 임계값은 골질이 D1–D2일 때의 기준이며, D3–D4에서는 다른 보정이 필요.

| **prospective** | [[implants/isq/lee-2011-isq-loading-protocol-prediction]] | ISQ ≥65 at placement → 조기/즉시부하 예측 가능 | 5개월, IT–ISQ 상관 확인 |

**합성 결론** [근거강함]: **즉시부하는 ISQ ≥70 + IT ≥35 Ncm 양조건 충족 시 안전.** ISQ ≥65 만으로도 조기부하 후보지만 IT 보조 필수. 신선 발치와에서는 ISQ가 인공적으로 낮게 나오므로 ITV를 1차 기준으로.

### 2.3 골질별 ISQ 기준값 (Bone Quality-Specific)

Lekholm-Zarb 분류 기반. [[implants/rosa-2024-do-dental-implants-bone-types]] 에서 발췌, [[implants/isq/bavetta-2024-isq-osstell-osseo-device-comparison]] prospective 데이터로 보강.

| 골질 (Lekholm-Zarb) | 부위 | ISQ 평균 범위 | 부하 결정 권고 |
|---|---|---|---|
| **Type I** (homogeneous compact) | 하악 전치부 | 72–80 | 표준 임계값 적용 (≥70 안전) |
| **Type II** (dense spongy + thick cortical) | 전치부 | 72–75 | 표준 임계값 적용 |
| **Type III** (dense spongy + thin cortical) | 상악 전치부 | 70–75 | 표준 임계값 적용, 경계선 주의 |
| **Type IV** (low-density + thin cortical) | **상악 구치부** | **58–65** | **표준 임계값 부적용**; ISQ ≥65로 하향 + 표면처리·바이오디자인·지연 부하 고려 |

**Type IV에서의 보정** [근거강함]:
- 실패 위험 type I 대비 1.92배 (Rosa 2024)
- 식립 시 ISQ 58–65이 정상 분포 — 70 미달이라고 자동 지연 결정 아님
- 친수성/생체활성 표면이 ISQ stability dip 제거 (Canullo 2024 prospective)
- 골밀도화 (Osseodensification, OD)가 type IV에서 가장 큰 ISQ 상승 (Fontes-Pereira 2023 SR)

**합성 결론** [claude해석]:
- **OD는 상악동 거상 (Sinus Lift) 동시식립에서 ISQ 상승이 가장 확실** (sr+ma 근거)
- 일반 식립에서 OD의 ISQ 우위는 **일시적 (mid-healing 한정)**, 3개월에 소멸 (RCT n=100)
- **생체활성·친수성·UV 광기능화 표면은 D3/D4에서 ISQ stability dip을 제거** → 조기부하 가능성 확대
- "표면처리로 임계값 자체가 내려간다"가 아니라 "임계값 도달 속도가 빨라진다"가 정확한 표현

### 2.5 Stability Dip — 부하 시점 결정의 함정

[합의수준 — Sennerby 2008 narrative 원형, Lee 2011 prospective 확인, Canullo 2024 prospective로 표면 의존성 확인]

```
ISQ
  │   ╭─── 1차 안정성 (기계적)
  │  ╭╯
  │ ╭╯    ← dip 구간 (2-4주)
  │╭╯╰──────── 2차 안정성 (생물학적)
  └──────────────────────→ 시간
```

| 인자 | Dip 정도 | 출처 |
|---|---|---|
| 즉시부하 Brånemark (혼합 골질) | **명확한 딥**: 70.4→66.4(30d)→68.8(90d); ~4 ISQ 하락 | Balshi 2005 prospective n=344 |
| 즉시부하, 연조직·구치부·여성 | 딥 더 크고 회복 더 느림 — 고위험 프로파일 | Balshi 2005 prospective |
| 전통 SLA, D3/D4 골 | 명확한 하락 (30일·45일 유의 감소) | Canullo 2024 prospective |
| 생체활성 나노표면, D3/D4 골 | **Dip 거의 없음** (T0=74.57 → T45=74.97) | Canullo 2024 prospective |
| Neoss ProActive 테이퍼, IT>40Ncm | Dip 없음 | Andreatta 2020 prospective |
| 친수성 테이퍼, 상악 구치부 | Dip 미보고, 6주 부하 가능 | Kim 2016 RCT |
| UV-PF 위축골 | Dip 없음, 6주에 +21.9 상승 | Hirota 2020 prospective |

**임상 시사** [claude해석]: "ISQ 트라젝토리는 표면처리가 결정한다." 식립시 ISQ가 같아도 표면에 따라 30–45일 ISQ는 5–10 차이 발생 가능. 부하 결정은 **식립시 단일값보다 2–4주 follow-up RFA가 더 신뢰성 높음** (Sennerby 2008 narrative).

## 3. 시나리오별 부하 결정 알고리듬

### 3.1 표준 임플란트 식립 (D1–D3 골)

```
식립 → IT 측정 → ISQ 측정
├─ ISQ ≥75 + IT ≥35 Ncm: 즉시부하 가능
├─ ISQ 70–74 + IT ≥35 Ncm: 조기부하 (6–8주)
├─ ISQ 65–69: 표준 부하 (3–4개월) + 6주 ISQ 재측정
├─ ISQ 60–64: 지연부하 + ISQ 모니터링; 표면처리·디자인 점검
└─ ISQ <60: 4–6개월 지연 또는 재평가
```

근거: Andersson 2019 (retrospective, n=745) · Wittneben 2023 (SR) · Oh 2008 (in-vivo) · Sennerby 2008 (narrative)

### 3.2 D4 골 (상악 구치부, 후방위축)

```
CBCT 치조정 피질골 두께 평가  ← Chatvaratthana 2017 prospective
└─ 피질골 <1mm: 추가 보강 고려

식립 → IT + ISQ
├─ ISQ ≥65 + IT ≥35 Ncm + 친수성/생체활성 표면: 6–8주 부하 가능
├─ ISQ 60–64 + 전통 SLA: 3–4개월 지연부하
├─ ISQ <60: OD·UV-PF·표면 강화 임플란트 재선택 검토
└─ 옵션: 골밀도화 (OD) 적용 시 baseline ISQ ~5–10 상승 기대
```

근거: Rosa 2024 (SR+MA) · Canullo 2024 (prospective) · Fontes-Pereira 2023 (SR) · Hirota 2020 (prospective)

### 3.3 즉시식립 (Immediate Placement, Type-1)

```
발치 후 즉시식립 → IT 측정이 1차 기준
├─ IT ≥35 Ncm: 즉시부하 후보 (ISQ 보조)
├─ IT 25–35 Ncm + ISQ ≥65: 임시보철 비기능부하 가능
└─ IT <25 Ncm: 지연부하 (12주 이상)

주의: 발치와 ISQ는 인공적으로 낮음 (~57 vs 치유골 ~72) — Bavetta 2019
경조직+연조직 이식 동반 시 ISQ 향상 + 심미 결과 우수 — Rondone 2024 SR+MA
```

근거: Wittneben 2023 (SR) · Bavetta 2019 (retrospective) · Rondone 2024 (SR+MA) · Balshi 2005 (prospective n=344): 즉시부하 30일 stability dip 최저점 → 60일 이내 프로토콜 변경 금지, 연조직·구치부·여성에서 더 심함

### 3.4 상악동거상술 동시식립

```
RBH (잔존골 높이) 평가
├─ RBH ≥6mm: TSFE + 동시식립 가능
│   ├─ OD 사용: ISQ ~66 at placement (vs 오스테오톰 ~55) — El-Ghobashy RCT
│   └─ 부하 시기: 3개월 ISQ 재측정 후 결정
├─ RBH 4–6mm: TSFE + OD 우선 (Shalash 2023 prospective: 평균 거상 4.42mm, 1년 100%)
└─ RBH <4mm: 측창식 거상 + 2단계 protocol

2단계 측창식 ISQ 궤적 (Ramirez-Fernandez 2017):
T0(식립) 63.8 → T2(3mo) 73.5 → T3(6mo) 74.65 [BBM 이식]
→ 6개월에 native bone과 동등 (Jelusic 2014)
```

근거: Starch-Jensen 2025 (SR+MA) · El-Ghobashy (RCT) · Ramirez-Fernandez 2017 · Jelusic 2014 · Shalash 2023 (prospective)

상세는 [[overviews/sinus-lift-isq-loading-timing]] 참조.

## 4. ISQ 단독 해석의 한계 — 보완 지표

[근거강함 — Tisci 2026 SR+MA · Huang 2020 narrative · Chatvaratthana 2017 prospective]

| 한계 | 보완 지표 |
|---|---|
| ISQ–IT 상관 r=0.44, I²>90% | IT 병행 측정 (≥35 Ncm) |
| ISQ는 치조정 피질골만 반영 (심부 6–9mm 무관) | CBCT 치조정 피질 두께 사전 측정 |
| 단일 ISQ 값은 trajectory 무시 | 식립·2주·6주 시계열 측정 |
| 골질이 ISQ 값을 좌우 | Lekholm-Zarb 분류로 임계값 조정 |
| 표면처리에 따라 dip 양상 다름 | 임플란트 시스템별 baseline 데이터 보유 |
| 신선 발치와에서 ISQ 인위적 저하 | ITV를 1차 기준으로 |

**핵심 원칙** [근거강함]: ISQ는 **하나의 보조 지표이지 단독 결정 지표가 아니다**. 임상 결정은 ISQ + IT + 골질 + 표면처리 + 시점 5요소 종합.

## 6. 미해결 쟁점 및 향후 근거 필요 영역

[claude해석]

1. **70이라는 컷오프의 절대성 검증 부족** — Andersson 2019 retrospective n=745가 가장 강한 근거이나 단일 시스템(Neoss). 다중 시스템 prospective RCT 부재.
2. **D4 골에서의 최적 임계값** — 58–65 정상범위와 70 표준 컷오프 사이 결정 가이드라인 없음. 표면처리·OD 조합 RCT 필요.
3. **ISQ 트라젝토리 패턴 분류** — Dip 유형 (없음·얕음·깊음·진행성 하락) 분류와 실패 위험 매트릭스가 아직 정립 안됨.
4. **즉시식립에서 ISQ 무용성** — Bavetta 2019에서 발치와 ISQ 57.6도 100% 성공. 발치와 전용 임계값 또는 ISQ 외 지표 (Periotest 등) 필요.

## 7. 결론 — 임상 적용 한 페이지 요약

**ISQ 부하 결정의 6대 원칙** [합의수준]:

1. **ISQ ≥70 at placement** = 표준 부하 안전선 (Andersson 2019 retrospective n=745)
2. **즉시부하**: ISQ ≥70 + IT ≥35 Ncm 양조건 충족 (Wittneben 2023 SR)
3. **6–8주 조기부하**: IT >40 Ncm 단독으로도 가능 (Andreatta 2020 prospective)
4. **D4 골**: ISQ 58–65이 정상 분포 — 70 미달이라고 자동 지연 결정 아님; 표면처리·OD로 보정 (Rosa 2024 SR+MA + Canullo 2024 prospective)
5. **신선 발치와**: ISQ 신뢰 불가, ITV ≥35 Ncm 우선 (Bavetta 2019 retrospective)
6. **상악동 동시식립**: OD 사용 시 식립시 ISQ ~10 상승 (Starch-Jensen 2025 SR+MA)

**ISQ를 절대 단독으로 결정에 사용하지 말 것.** IT, 골질, 표면처리, 시점을 함께 평가.
