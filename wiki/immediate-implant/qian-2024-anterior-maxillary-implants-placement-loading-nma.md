## 한줄요약
SR+NMA (Semmelweis): 상악 전치부 단일 임플란트 — type 1/2/3/4 placement × immediate/early/conventional loading 조합 — 생존율·MBL·심미·PROMs 비교; 다양한 프로토콜 직접·간접 비교.

## Summary
Systematic review + network meta-analysis comparing all implant placement and loading protocol combinations for single anterior maxillary implants. Outcomes include survival, MBL, PES, patient satisfaction.

## Key Contributions
- 핵심 데이터는 Results 섹션 참조.
- 자세한 contribution은 원문 Introduction/Discussion 참고.

## Methodology
PRISMA-NMA. Bayesian/frequentist NMA framework. SUCRA ranking. Risk of bias.

## Results
(본문 기반: type 1A (immediate + immediate loading)와 type 1B (immediate + delayed loading) 모두 적절한 case selection 하에서 conventional protocols와 동등; 그러나 type 1A는 술자/case 의존도 높음.)

## Clinical Relevance
상악 전치부 단일치에서 timing 결정 시 NMA 근거: 일반적으로 type 2/3 + conventional/early loading이 안전; type 1A는 ITI Hamilton 2023 selection criteria 충족 시 적용.
