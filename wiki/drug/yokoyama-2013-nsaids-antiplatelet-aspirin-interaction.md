## 한줄요약
인비트로 (9종 NSAIDs, 임상 정상혈장농도): 이부프로펜(-48.1%)·메페남산(-22.7%)만 아스피린 앞 복용 시 항혈소판 효과 유의하게 감소; 나프록센·플루르비프로펜은 자체 항혈소판 효과 있으나 아스피린 간섭 없음.

## Summary
This in vitro Japanese study screened nine clinically available NSAIDs for their ability to interfere with aspirin's antiplatelet effect, using steady-state plasma concentrations derived from published pharmacokinetic data. Human platelet-rich plasma (PRP) collagen-induced aggregation was measured under four conditions: NSAID alone, aspirin alone, aspirin-before-NSAID, and NSAID-before-aspirin. Only ibuprofen (48.1% reduction in aspirin's effect) and mefenamic acid (22.7% reduction) significantly interfered when given before aspirin. Critically, none of the nine NSAIDs altered aspirin's effect when aspirin was administered first. Naproxen and flurbiprofen showed their own independent antiplatelet effects at therapeutic plasma concentrations but did not block aspirin's COX-1 acetylation.

## Key Contributions
- Systematic in vitro comparative screen of nine NSAIDs at therapeutic plasma concentrations
- Identifies ibuprofen as the primary aspirin antagonist (48.1%) and mefenamic acid as secondary (22.7%)
- Confirms timing rule: NSAID-after-aspirin is safe for all nine agents tested
- Distinguishes two mechanisms: (1) blocking aspirin acetylation (ibuprofen/mefenamic acid) vs. (2) independent reversible COX-1 antiplatelet activity (naproxen/flurbiprofen)
- Confirms safety of celecoxib, diclofenac, loxoprofen, indomethacin, etodolac, and meloxicam with concurrent aspirin

## Methodology
- In vitro human PRP; collagen-induced platelet aggregation
- NSAIDs at steady-state Cmax derived from standard clinical doses (Japanese pharmacokinetic interview forms)
- Aspirin at EC50 concentration (in vitro)
- Four conditions per NSAID: NSAID alone / aspirin alone / aspirin→NSAID / NSAID→aspirin

## Results
| NSAID | % Reduction of Aspirin Effect (before aspirin) | Independent Antiplatelet Effect |
|---|---|---|
| Ibuprofen | **48.1%** (significant) | — |
| Mefenamic acid | **22.7%** (significant) | — |
| Naproxen | Not significant | Yes (own effect) |
| Flurbiprofen | Not significant | Yes (own effect) |
| Celecoxib | None | None |
| Loxoprofen | None | None |
| Indomethacin | None | None |
| Diclofenac | None | None |
| Meloxicam | None | None |

- No NSAID interfered when aspirin was given first (all conditions: aspirin→NSAID = no effect)
