## 한줄요약

## Summary
International Delphi consensus developing a core outcome set (COS) for dental antibiotic stewardship intervention research. Patient, academic, and clinician representation. Establishes which outcomes future stewardship trials should report as standard, enabling comparison and meta-analysis across interventions.

## Key Contributions
- First international COS for dental antibiotic stewardship.
- Multistakeholder (patient + clinician + academic) input.
- Standardizes outcome reporting in future stewardship trials.

## Methodology
Delphi consensus; outcomes sourced via literature review; ≥30 international participants; ≥70% "critical for inclusion" threshold for COS inclusion.

## Results
[합의수준] Core outcome set defined; specific items in paper.

## Limitations
- Process-focused, not direct prescribing data
- Local adaptation required for non-English-speaking contexts
