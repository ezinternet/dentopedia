## Summary
CTS 진단 어려움 mini review. Longitudinal fracture 5 type 분류 context 정리 + multi-modal diagnostic 강조.

## Longitudinal Fracture 5 Type

| Type | 특징 |
|---|---|
| Craze line | 표면, enamel only, 무증상 |
| Fractured cusp | 교두 partial 파절 |
| **Cracked tooth (CTS)** | Incomplete fracture, pulp까지 진행 가능 |
| Split tooth | Complete fracture, 치아 둘로 분리 |
| Vertical root fracture | 치근 종방향 파절, 발치 가능 |

## Diagnostic Challenge
- Single tool 불가 — visual, bite test, transillumination, dye, CBCT 종합
- Symptom variability — 일관된 호소 어려움
- 영상 (X-ray, CBCT) 한계 — 미세 crack 시각화 어려움

## Strengths and Caveats
