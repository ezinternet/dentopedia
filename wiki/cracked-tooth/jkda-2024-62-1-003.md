## Diagnostic Tools

| 도구 | 장점 | 한계 |
|---|---|---|
| Visual + transillumination | 빠름, 비침습 | 깊은 crack 누락 |
| Dye staining (methylene blue) | 시각화 | 시술 침습 |
| Bite test (Tooth Slooth) | 환자 호소 재현 | 위양성·위음성 |
| CBCT | 3D | Crack은 직접 시각화 어려움 |
| 진단 결정 검사 | endo 진입 후 직접 확인 | 비가역적 |

## Clinical Framework
1. Patient history (sharp pain on chewing release)
2. Visual + transillumination
3. Bite test (cusp별 sequential)
4. CBCT (deep crack·root involvement 의심)
5. Tentative diagnosis → endo 진입 후 final confirmation
