## 한줄요약
Ex-vivo diagnostic study (n=147 extracted maxillary molars): CBCT sensitivity 0.88 / specificity 0.88, statistically equivalent to 16× microscope and to the sectioning reference standard; loupe and naked-eye perform clearly worse.

## Summary
Comparative diagnostic-accuracy study of four MB2-detection methods on 147 extracted maxillary molars. Index tests: naked-eye, 3.5× loupe, 16× operating microscope, and CBCT. Reference standard: horizontal sectioning of each tooth (cervical, medial, apical thirds) inspected by an endodontist. The 16× microscope and CBCT were statistically equivalent to the reference standard, supporting both as high-yield clinical tools; loupes and unaided vision under-detected MB2.

## Key Contributions
- Same-tooth direct comparison of four detection modalities against a sectioning reference standard
- CBCT diagnostic metrics: sensitivity 0.88, specificity 0.88, PPV 0.84, NPV 0.91
- Microscope (16×) matched CBCT and the reference standard — supports DOM as the chair-side anchor when CBCT is unavailable

## Methodology
- Design: ex-vivo diagnostic accuracy study
- Sample: 147 extracted permanent maxillary molars
- Index tests: naked-eye, loupe (3.5×), DOM (16×), CBCT
- Reference standard: horizontal sectioning + endodontist inspection
- Outcomes: sensitivity, specificity, PPV, NPV per modality

## Results
- **CBCT**: sens 0.88, spec 0.88, PPV 0.84, NPV 0.91
- **Microscope (16×)**: NS difference vs reference standard (highest agreement)
- **Loupe (3.5×)** and **naked-eye**: clearly inferior detection
- Clinical implication: when a maxillary molar endo is undertaken without DOM and without CBCT, expect under-detection of MB2

## Limitations
- Ex-vivo: no chamber blood/restoration/calcification noise
- Operator blinding details sparse
- CBCT voxel size not reported
- Reference standard (sectioning) itself can miss apical-third MB2
