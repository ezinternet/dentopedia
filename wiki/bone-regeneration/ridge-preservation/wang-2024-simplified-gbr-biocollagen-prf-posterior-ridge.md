## One-line Summary

Retrospective study (n=44) comparing simplified GBR (Bio-collagen + PRF membrane) vs classical GBR (Bio-Oss + Bio-Gide) for posterior Seibert Class I ridge defects: HW gain 1.50 vs 1.83 mm at 6 months; comparable Type A buccal contour predominance.

## 한줄요약

후향적 연구(n=44)에서 후방 Seibert Class I 결손에 단순화 GBR(Bio-collagen+PRF막) vs 고전적 GBR(Bio-Oss+Bio-Gide) 비교: 수평폭경 증가 1.50 vs 1.83mm(6개월), 협측 윤곽 Type A 우세 유사.

## Summary

For minor Seibert Class I ridge defects in the posterior region (horizontal deficiency, normal height), extensive augmentation with expensive xenograft + collagen membrane may be unnecessary. This retrospective study tested a **simplified GBR** approach using **Bio-collagen (BC) + autologous PRF membrane** as a cost-effective, biologic alternative to classical Bio-Oss + Bio-Gide GBR.

At 6 months (CBCT), both approaches produced meaningful horizontal width gain: simplified GBR 1.50 mm vs classical 1.83 mm. Buccal ridge contour (Type A = good, B = moderate, C = poor) was predominantly Type A in both groups. The 0.33 mm gap in HW gain is likely clinically minor for the target indication (minor posterior ridge defects pre-implant). The study introduces a 3-type buccal contour morphology classification (Type A/B/C) as a qualitative outcome tool.

## Key Contributions

1. **PRF membrane as substitute** for resorbable collagen barrier in minor posterior ridge defects — avoids Bio-Gide cost while maintaining augmentation effect
2. **Buccal contour classification (A/B/C)** introduced as morphological outcome measure for ridge augmentation
3. **HW gain 1.50 mm** with simplified GBR — statistically significant vs baseline; 0.33 mm less than classical GBR
4. Type C (poor contour) lower in simplified vs classical group (5% vs 14%) — trend favoring simplified

## Methodology

- Retrospective; n=44 (22 each)
- **Simplified GBR**: Bio-collagen graft + PRF membrane barrier
- **Classical GBR**: Bio-Oss + Bio-Gide collagen membrane
- **Outcome**: CBCT at 6 months; HW (horizontal width) and buccal HW; buccal ridge contour Type A/B/C
- **Population**: Posterior Seibert Class I ridge defects (buccal horizontal deficiency, normal height)

## Results

| Outcome | Simplified GBR (n=22) | Classical GBR (n=22) |
|---|---|---|
| HW gain | 1.50 mm | 1.83 mm |
| Contour Type A | 14/22 (64%) | 11/22 (50%) |
| Contour Type B | 7/22 (32%) | 8/22 (36%) |
| Contour Type C | 1/22 (5%) | 3/22 (14%) |
