## 한줄요약
Review (HKU OMFS, Diagnostics 2022): CBCT를 통한 상악동 건강·병변 평가의 종합 review — 정상 해부·SMT 임계값·AP/MRC/sinusitis/odontogenic/tumor pathology 영상 매핑. Sinus pathology의 imaging 결정 가이드.

## Summary
HKU 그룹의 narrative review. CBCT가 sinus pathology 진단의 1차 도구로 자리잡은 현 상황에서 정상-병리 영상 표준 정리. AP/MRC 결정 트리의 0단계 (정확한 진단부터)를 다룸.

## CBCT 평가 항목

| 항목 | 정상 | 병리 시사 |
|---|---|---|
| **Sinus Mucosal Thickness (SMT)** | <2 mm | ≥2 mm |
| **Antral Pseudocyst (AP)** | 없음 | Dome-shaped, base on sinus floor |
| **MRC** | 없음 | 비슷한 dome — size 추적 필요 |
| **Mucocele** | 없음 | Sinus 전체 침범 가능 |
| **Septa** | 47-58% (population 의존) | 술식 위험 (Yang 2024 OR 4.03) |
| **Ostium patency** | 개방 | 폐쇄 → sinusitis 위험 |
| **Schneiderian membrane** | Intact | Perforation·thickening |

## Key Findings
- SMT ≥2mm가 pathology 의심 임계값 (Munakata 2021 cohort: 25%가 ≥2mm)
- AP·MRC 영상 차이 미세 — 조직학으로 확진
- Mucocele은 sinus 전체 침범 가능 → ENT 협진 강제
- Implant 식립 전 CBCT는 sinus pathology 평가의 표준

## Decision Tree Position
**결정 트리 0단계 — 정확한 진단**:
1. CBCT로 SMT·lesion 평가
2. <2mm SMT + 무증상 → 일반 sinus lift 진행
3. ≥2mm SMT + lesion → Nosaka 2024 framework로 감별
4. 광범위 침범 또는 침습성 → ENT 협진
