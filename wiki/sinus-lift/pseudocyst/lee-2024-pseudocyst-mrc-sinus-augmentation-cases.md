## One-line Summary
Case series (Seoul National Univ OMFS, J Craniofac Surg 2024, n=3): proposes a sinus-pathology decision tree — type 1 (pseudocyst, PsC) → spontaneous drainage, type 2 (MRC <20 mm) → aspiration, type 3 (MRC >20 mm) → enucleation — each followed by augmentation; all 3 implants integrated at 1-year, though the 20 mm cut-off is hypothetical and case-derived.

## 한줄요약
Case series (서울대 치병원 OMFS, J Craniofac Surg 2024, n=3): sinus pathology를 **type 1 (PsC), type 2 (MRC <20mm), type 3 (MRC >20mm)**로 분류, 각각 spontaneous drainage·aspiration·enucleation 차등 적용 — 의사결정 트리 직접 case 시각화.

## Summary
서울대 OMFS 그룹의 case series로, sinus pathology 처치의 **명시적 결정 트리 framework**를 3 case로 보여준다. 20mm를 cut-off로 한 분류는 임상 직관에 부합하지만 cut-off의 정량적 근거는 가설적.

## Decision Tree (본 논문 핵심)

| Type | Pathology | Size | 술식 |
|---|---|---|---|
| 1 | Antral Pseudocyst (PsC) | — | Spontaneous drainage → augmentation |
| 2 | Mucous Retention Cyst (MRC) | <20 mm | Aspiration → augmentation |
| 3 | MRC | >20 mm | Enucleation → 회복 → augmentation |

## Results
- 3 case 모두 성공적 implant 식립·정착
- Subtype별 술식 차등이 framework로 작동
- 각 case의 1년 추적 OK

## Clinical Relevance
- **본 framework의 가치**: 술자에게 결정 알고리즘 제공 — 술식 일관성 ↑
- **한계**: 20mm cut-off는 본 case에서 가설적 — Fu 2025 (3-arm cohort, n=107)는 ≤20mm 전체를 한 군으로 다뤄 cut-off 검증 못함
- **임상 활용**: 외래 진단 시 type 분류 → 술식 일관성 확보 → 환자 동의서 작성도 용이

## Strengths and Caveats
[case-report] n=3 — 통계 결론 불가
[합의수준] Subtype별 차등 처치는 임상 직관에 부합
[claude해석] 20mm cut-off는 본 case 이후 cohort/RCT로 validate 필요
[미검증] Type 1에 spontaneous drainage 적용 가능성 — case 의존적, 어떤 PsC가 자연 배출되는지 예측 어려움
