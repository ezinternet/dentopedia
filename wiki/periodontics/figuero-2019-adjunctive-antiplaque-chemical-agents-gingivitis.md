## One-line Summary

SR + NMA (53 RCTs, n=8,457) comparing anti-plaque agents for gingivitis: essential oils rank highest for GI reduction; CHX ≥0.10% and triclosan-copolymer are effective; CHX efficacy threshold at 0.10% concentration.

## 한줄요약

SR + 네트워크 MA (53 RCT, 8,457명) — 가글류 치은염 감소 순위: 에센셜오일 > CHX ≥0.10% ≈ triclosan; CHX 유효 농도 임계값 0.10%로 확인.

## Summary

6개월 home-use RCTs를 대상으로 화학적 항균 가글·치약의 치은염 감소 효능을 NMA로 비교했다. 직접 비교 RCT가 없는 제품 간 간접 비교를 제공한다는 점에서 중요하다.

**가글 순위 (GI 감소)**:
1. **에센셜오일 (Essential oils)**: 가장 큰 GI 감소 — Listerine류 (thymol·eucalyptol·menthol·methyl salicylate)
2. **Triclosan-copolymer**: CHX와 유사
3. **CHX ≥0.10%**: 에센셜오일·triclosan과 통계적 차이는 없으나 수치상 낮음
4. **CPC >0.05%**: 유효

**CHX 농도 임계값**: CHX ≥0.10% 농도에서 GI 유의 감소; 0.05% 이하는 데이터 부충분.

**치약 비교**: 모든 활성 치약 간 차이 없음; sanguinarine·baking soda는 최하위.

**임상적 해석**:
- 에센셜오일 가글은 CHX와 효능 유사하면서 착색 side effect 없음 → 장기 사용 시 대안 고려 가능
- 하지만 연구 간 이질성·NMA 가정(transitivity) 한계 고려 필요

## Key Contributions

1. **가글 간 순위** 제공 — CHX를 에센셜오일·triclosan·CPC와 맥락 비교
2. **CHX 농도 임계값 ≥0.10%** 확립 — 0.12%·0.2% 등 표준 농도 근거
3. 53 RCT NMA — 직접 head-to-head 없는 제품 간 간접 비교

## Methodology

- **Selection**: 6개월 home-use RCTs; GI 측정
- **Studies**: 19 mouthrinse (n=5,775) + 32 dentifrice (n=2,682)
- **Scales**: Löe & Silness GI, modified GI, gingival severity index
- **NMA protocol**: Direct + indirect comparison; SMD

## Results

**Mouthrinse NMA 순위 (GI)**:

| Agent | Efficacy rank |
|---|---|
| Essential oils | 1 (highest) |
| Triclosan-copolymer | 2 |
| CHX ≥0.10% | 3 |
| CPC >0.05% | 4 |
| Placebo | Last |

- Active agent 간 직접 비교에서 통계적 유의차는 대부분 없음
- Vs placebo는 상위 4개 모두 유의
