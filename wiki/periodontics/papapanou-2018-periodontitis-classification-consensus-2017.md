## One-line Summary
2017 World Workshop consensus report (international Delphi expert consensus) establishing the new periodontitis classification: Staging I–IV (severity/complexity via CAL, RBL, tooth loss, furcation, mobility) plus Grading A–C (progression rate, systemic modifiers, treatment response), replacing the 1999 system and eliminating the aggressive periodontitis category.

## Summary
2017 World Workshop consensus establishing the new Stage (I–IV) and Grade (A–C) classification for periodontitis.

## Key Contributions
- Stage I–IV: severity (CAL, RBL, tooth loss) + complexity (furcation, mobility, vertical bone loss)
- Grade A–C: progression rate, systemic modifiers (diabetes, smoking), treatment response
- Stage IV: mastication compromise; aggressive periodontitis category eliminated

## Methodology
Systematic literature search + international Delphi expert consensus.

## Results
New classification replaces 1999 system. Stage IV: tooth loss attributable to periodontitis + mastication dysfunction. Grading incorporates systemic disease impact.
