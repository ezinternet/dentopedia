## 한줄요약
증례보고: 하치조신경 노출 → dexamethasone 국소도포 + prednisolone 점감 + NSAIDs → 36일 내 neuropraxia 완전 회복

## Summary
하악 제3대구치 발치 중 하치조 신경혈관 다발 노출 → neuropraxia 진단 → **dexamethasone 국소도포 + prednisolone 점감법 + NSAIDs** 병용 치료로 36일 내 완전 회복한 증례.

## 치료 프로토콜 (즉각 대응 순서)

1. **수술 중**: Dexamethasone 2 mL → 노출 신경 표면에 **5분간 국소 도포**
2. **경구 prednisolone 점감** (10일):
   - Day 1–2: 25mg → Day 3–4: 20mg → Day 5–6: 15mg → Day 7–8: 10mg → Day 9–10: 5mg
3. **NSAIDs**: Ibuprofen 400mg × 3회/일
4. **항생제**: Cefaclor 250mg
5. **냉각요법**: 즉시 적용 (2차 신경손상·신경종 예방)

## 회복 경과

| 시기 | Brush test (Rt/Lt) | 상태 |
|---|---|---|
| 술 후 5일 | 100/70 | 이감각증 |
| 술 후 8일 | 100/70 | 호전 시작 |
| 술 후 12일 | 100/90 | 현저히 감소 |
| **술 후 36일** | **100/100** | **완전 회복** |

## 임상 핵심 포인트
- 신경혈관 다발 노출 시 **국소 dexamethasone이 전신 투여보다 효과적** (Suslu et al.)
- 흡수 시간 확보가 중요 — 세척으로 씻겨나가면 효과 감소
- 이감각증이 **3개월 내 미회복 시** → 외과적 개입 고려
- 표준화 프로토콜 없음 — 현재까지 근거 미약, 임상 판단 필요

## 신경손상 분류
```
Neuropraxia → 자연회복 가능 (본 증례)
Axonotmesis → 회복 가능하나 시간 소요
Neurotmesis → 외과적 재건 필요
```
