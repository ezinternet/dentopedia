## 한줄요약
증례보고: UV 광기능화(접촉각≈0°, BIC≈100%) → stability dip 없이 ISQ 급상승 → 1–4주 조기부하 성공

## Summary
밀폐형 UV 처리 임플란트 증례보고. UV 광기능화로 초친수성(접촉각≈0°)·양전하·저탄소 표면 구현 → **stability dip 없이** ISQ 식립 2주 내 급상승(예: 69→81) → **식립 1-4주 후 조기부하 성공**.

## UV 광기능화 효과

| 특성 | 일반 Ti | UV 처리 Ti |
|---|---|---|
| 접촉각 | ~70° | **≈0°** |
| 표면 탄소 | 60-75% | **<20%** |
| 표면 전하 | 음전하 | **양전하** |
| BIC (동물) | 55% | **≈100%** |

## ISQ 추이 (Case #1, 발치와 즉시식립)

| 시점 | #36 | #37 |
|---|---|---|
| 식립일 | 69 | 74 |
| 1주 | 73 | 75 |
| **2주** | **81** | **81** |
| 4주 (부하) | 80 | 80 |

→ Stability dip 없음; 조기부하 후 안정 유지

## Clinical Relevance
- UV 처리 → 골유착 신속화 → 조기부하 타이밍 앞당김
- ISQ ≥ 70 도달 후 즉시/조기 부하 고려
- 연질골·즉시식립 고위험 케이스에서 안전 마진 향상
- 증례보고 수준 — 대규모 RCT 검증 필요
