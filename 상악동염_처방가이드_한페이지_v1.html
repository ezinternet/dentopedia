치성 상악동염 (OMS) 처방·치료 한 페이지

v1 · 2026-05-20
핵심: 치성 원인 제거 + 부비동 배농. 1차 Augmentin ± Metronidazole 2~4주 + 비강 corticosteroid·세척 → 무효 시 치과 + ESS 결합. 단독 ESS는 29% 불응.

1 처방 (1차)

약물용법

Metronidazole (혐기성)500mg tid 병용
페니실린 알러지Clindamycin 300mg qid
또는 Moxifloxacin 400mg qd
보조비강 corticosteroid + 세척
기간비수술 단계 2~4주
주의: 알러지·신기능·임신·항응고제·메트로니다졸-알코올 금기 확인. Zirk: amoxicillin 단독 내성↑ → β-lactamase inhibitor 병용 필수.

→
② 치성 원인 치료근관·치주·발치·임플란트 제거·OAF 봉합

③ 분기 판단호전+CT 낮음 = 치과단독 / 잔존+OMC폐쇄 = 치과+ESS

5 의사결정 체크리스트

  • 임플란트·근관치료·발치 직후 → iatrogenic OMS 우선
  • Zn-oxide sealer 과충전 → fungal ball 유발
  • 치근 sinus 근접 순: 2M > 1M > 3M > 2PM > 1PM
  • 편측 비폐색 + 화농비루 + 후비루 + 안면통 2~4주 무효 → ENT 조기 의뢰
  • CBCT가 일반 CT보다 치아-부비동 관계 우수
  • Schneiderian membrane ≥5mm 비후 → 약물 단독 한계
  • 치성 원인 명확 + sinus 부담 가벼움 → 치과 단독 가능
