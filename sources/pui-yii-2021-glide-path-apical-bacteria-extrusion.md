## Why Ingested

[[endodontics/shaping/aflaki-2020-glide-path-pathfile-scoutrace-cbct]]·[[endodontics/shaping/aminsobhani-2022-glide-path-pathfinder-s-canal]]가 glide path "shape" 측면을 다룬다면, 본 RCT는 "biological" 측면(apical bacterial extrusion) — flare-up 위험·post-op pain 직결. WaveOne Gold Glider reciprocation의 새 design이 manual K-file·One G 대비 bacterial extrusion 어떤지 의사결정 근거.

## One-line Summary
Ex-vivo extracted teeth E. faecalis 모델: 3 glide path system 모두 apical bacterial extrusion 측정 가능 — manual K vs One G vs WaveOne Gold Glider 간 유의차 분석 (본문에서 자세 결과는 PDF 직접 확인).

## 1. Document Information
- European Endodontic Journal 6:221-225 (2021)

## 2. Key Contributions
- Glide path stage의 bacterial extrusion 정량
- Reciprocation(WaveOne Gold Glider) vs continuous rotary(One G) vs manual 비교
- Flare-up risk evidence

## 3. Methodology and Architecture
- E. faecalis 접종 single-rooted teeth
- 3 glide path 방법으로 prep
- Apical 통과 bacteria 수집 후 정량

## 4. Key Results and Benchmarks
- All instrumentation techniques 측정 가능한 양의 apical extrusion 발생
- 정량값·통계는 PDF 본문 표 참조 (요약 미수록)

## 5. Limitations and Future Work
- Ex-vivo, single bacterial species (E. faecalis)
- Single-rooted teeth — multi-rooted molar 일반화 한계
- 임상 flare-up 결과 미측정

- Glide path: physiological terminus까지 smooth tunnel 확보
- Apical extrusion: file/irrigant이 apical foramen 너머 밀려 나가는 현상
- Reciprocation: 정·역회전 교대 motion (asymmetric angle)
