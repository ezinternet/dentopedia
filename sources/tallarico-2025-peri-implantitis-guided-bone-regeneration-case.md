## One-line Summary
Case report (44F): peri-implantitis + periodontitis → implant + tooth removal → 9mo healing → CAD/CAM titanium mesh GBR (autograft+bovine) → fully guided 2-implant placement (hydrophilic SOI surface) → monolithic zirconia — successful digital rehabilitation workflow.

## 1. Document Information
- Journal: Reports 8(3):118 (July 2025)
- DOI: 10.3390/reports8030118
- Design: Case report, single patient, posterior mandible

## 2. Case Summary
- Patient: 44-year-old female
- Problem: failing implant + adjacent tooth (advanced peri-implantitis + periodontitis)
- Workflow:
  1. Implant + tooth extraction → healing
  2. CBCT + intraoral scan → virtual bone regeneration planning
  3. CAD/CAM customized titanium mesh fabrication
  4. GBR: autogenous bone + anorganic bovine (A-Oss) + titanium mesh
  5. 9 months healing → mesh removal
  6. Fully guided implant placement (OneGuide system) — 2 hydrophilic SOI implants
  7. Soft tissue graft
  8. Final: monolithic zirconia fixed prosthesis

## 3. Key Concepts
- Digital workflow: IOS + CBCT + virtual planning → CAD/CAM titanium mesh
- Prosthetically driven: implant position planned from prosthetic outcome backward
- Hydrophilic surface (SOI): promotes osseointegration in compromised regenerated bone
- Titanium mesh for vertical GBR: maintains space, allows vascularization

## 4. Limitations
- Single case report (n=1)
- No control; no survival/ISQ data reported
- Short post-restoration follow-up described
