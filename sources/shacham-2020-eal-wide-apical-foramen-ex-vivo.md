## One-line Summary
Ex vivo (10 teeth, foramen widened to 0.6–0.8mm): EAL accuracy and stability both improve as file size approaches foramen diameter; introduces "stability" as a new accuracy parameter; file matching to canal diameter recommended.

## 1. Document Information
IDF Medical Corps, Israel. BDJ Open 2020. Ex vivo study addressing EAL accuracy in wide apical foramina. Novel contribution: introduces EAL measurement "stability" (constant reading ≥5 seconds) as a validity parameter alongside accuracy.

## 2. Key Contributions
- New parameter: **stability** of EAL reading (constant ≥5 sec) correlates significantly with accuracy
- Key finding: as file size approaches foramen diameter → accuracy AND stability both improve
- SAF (self-adjusting file) provides accurate and stable EAL readings in wide foramina
- Wide foramen (≥0.6mm) with undersized file → inaccurate, unstable EAL readings

## 3. Methodology
- 10 straight-canal teeth (single-rooted + palatal/distal roots of molars)
- Sequential foramen widening to 0.6, 0.7, 0.8mm (#60 SS K-file)
- Alginate mold; AFL determined under DOM
- Sequential K-file sizes + SAF measured at each widening stage
- Stability: reading held constant ≥5 seconds
- Statistical: Pearson correlation for accuracy-stability relationship

## 4. Key Results
- Accuracy ↑ as file-to-foramen diameter difference ↓ (statistically significant)
- Stability ↑ as file-to-foramen diameter difference ↓ (statistically significant)
- Accuracy and stability correlate significantly (p<0.05)
- SAF: both accurate and stable in 0.6–0.8mm wide foramina
- Wide foramen + small file = unstable + inaccurate → clinical risk of long measurement

## 5. Limitations
- n=10 teeth only
- Ex vivo — no tissue fluid, PDL; alginate model
- Only straight canals; curved canals not tested
- SAF is a specialized file not universally available

## 7. Glossary
- **AFL**: Apical Foramen Length (true canal length to foramen)
- **SAF**: Self-Adjusting File (NiTi, adapts to canal diameter)
- **Wide apical foramen**: >0.6mm (often after overinstrumentation, resorption, or immature apex)
- **Stability**: EAL reading maintained constant for ≥5 seconds
