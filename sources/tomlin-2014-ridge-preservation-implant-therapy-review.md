## One-line Summary
발치 후 치조제 보존술 문헌 고찰: 자연 치유 시 수직 소실 ~1mm·수평 소실 4-6mm(50%), 발치 후 3개월 내 대부분 발생; 발치와 보존술은 혈병 단독보다 모두 우수하나 단일 최우수 술식 없음.

## 2. Key Contributions
- 발치 후 자연 치유의 수직·수평 골소실 수치 종합 제시
- 치조제 보존술 기법별 근거 정리 (골이식재, 차폐막, 생물학적 제제)
- **발치 후 3개월 내 골소실의 2/3 발생** 강조 → 적기 개입의 중요성
- 단일 우월 기법 없음; 모든 수용 가능 기법이 혈병보다 우수함 결론

## 3. Key Results

### 자연 치유 시 골소실 (Schropp 등)
| 지표 | 소실량 |
|---|---|
| **수직 골소실** | 0.5–0.9 mm (12개월) |
| **수평 골소실** | 4.5–6.1 mm (12개월; 원래 폭의 ~50%) |
| **3개월 내 발생** | 전체 소실의 **2/3** |

- 구치부 > 소구치부에서 소실 더 큼
- 치조정 높이는 인접치 부착 수준까지 도달하지 못함

### 발치 후 치유 단계 (Ohta; 원숭이)
1. 육아 단계 (~5일): 소켓 하부에서 육아조직 형성
2. 초기 혈관형성 단계 (1주): 혈병 수축, 소주골 형성 시작
3. 신생골 형성 단계 (2주): 골양질(osteoid) 검출 가능
4. 골성장 단계 (4-5주): 소켓 2/3 충전
5. 골재구성 단계 (6주~): primary → secondary spongiosa 전환

### 치조제 보존술 결론
- 골이식재 단독, 골이식재+차폐막, 차폐막 단독: **모두 혈병 단독보다 우수** (RCT 근거)
- 단일 최우수 기법 없음 (근거 수준 이질성)
- 생물학적 제제(PRP, 성장인자): 추가 이점 불분명

## 4. Clinical Implications
- 발치 즉시 치조제 보존술 시행이 향후 임플란트 식립 조건 개선
- 수평 소실이 수직보다 현저함 → 임플란트 위치 및 보철 설계에 영향
- 발치 후 3개월이 임계점: 이 시기 전 개입이 가장 효과적
- 수술자가 익숙한 검증된 기법 선택 권고

## 5. Related Work

- Chappuis 2013 (this wiki): 인간 3D CBCT 분석, 박벽형 7.5mm 소실
- Covani 2010 (this wiki): 단일치 발치 후 치조제 재형성 패턴

## 7. Glossary
- **Ridge preservation (치조제 보존술)**: 발치 직후 소켓에 이식재/막 삽입하여 골소실 최소화
- **Primary spongiosa**: 새로 형성된 미성숙 해면골 (woven bone)
- **Secondary spongiosa**: 성숙한 층판골로 재구성된 해면골
- **Horizontal resorption**: 협-설 방향 골소실 (발치 후 지배적 소실 양상)
