## 1. Document Information
- Authors: Wells LK, Drum M, Nusstein J, Reader A, Beck M
- Affiliation: Ohio State University
- Type: Prospective randomized double-blind RCT
- Journal: J Endod 2011;37(12):1608-1612

## 2. Key Contributions
1. RCT in a specific high-pain endodontic indication (necrosis + apical lesion + moderate-severe pain).
2. Tests the ibuprofen + APAP combination vs ibuprofen alone.
3. Establishes combination superiority in this real-emergency context.

## 3. Methodology
- n=71 adult patients with maxillary/mandibular symptomatic necrosis + apical radiolucent area + moderate-severe pre-op pain
- Emergency debridement performed
- Randomized to ibuprofen 600mg q6h vs ibuprofen 600mg + APAP 1000mg q6h post-op
- Outcomes: postoperative pain VAS, escape medication use

## 4. Key Results
[근거강함] Ibuprofen + APAP combination → less pain + less escape medication usage vs ibuprofen alone.

## 5. Limitations
- Small RCT (n=71)
- Single center (OSU)
- Doesn't compare to alternative regimens (codeine combos, etc)
