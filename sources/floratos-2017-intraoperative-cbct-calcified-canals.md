## One-line Summary
3-case series: intraoperative CBCT used to localize calcified/obliterated canals when clinical exploration fails — gutta-percha reference points placed, CBCT taken, distance to actual canal orifice measured in 3D.

## 1. Document Information
University of Pennsylvania + University of Athens. Case report / clinical technique description.

## 2. Key Contributions
- Novel protocol: gutta-percha points placed at estimated orifice → intraoperative CBCT → 3D distance calculation to true orifice
- Allows precise directional adjustment when clinical methods fail
- Third case: only one orifice locatable due to severe obliteration

## 3. Methodology
- 3 cases with clinically unlocatable canal orifices
- GP points placed → intraoperative CBCT → 3D distance measurement
- Cases 1-2: biplane distance calculation; Case 3: single orifice only found

## 4. Key Results
- CBCT provides 3D road-map for obliterated canal navigation
- Measurable distance in all planes → targeted dentin removal
- Clinical feasibility demonstrated; no perforation reported

## 5. Limitations
- n=3 cases
- Radiation exposure of intraoperative CBCT
- Operator skill-dependent

## 6. Related Work
- Mohammadi 2016 — CBCT role in missed canal prevention
- Dioguardi 2024 — access cavity design for canal localization

## 7. Glossary
- **CBCT**: Cone Beam Computed Tomography
- **Obliterated canal**: Canal narrowed/closed by secondary/tertiary dentin
- **GP point**: Gutta-percha point (radiopaque reference marker)
