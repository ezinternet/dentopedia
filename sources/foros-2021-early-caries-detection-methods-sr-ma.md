초기 우식 탐지 방법(DIAGNOdent, ICDAS-II, 방사선 등) SR+MA: DD Pen 민감도 0.81–0.89 최고, 교합면 기준 방사선 민감도 0–0.29 최저.

## 2. Key Contributions
- 2000–2019년 조기 우식 탐지 진단정확도 연구 51개 포함
- 영구치·유치 교합면·인접면 분리 분석
- 조직학적 기준 대비 각 방법별 Se/Sp 정량

## 3. Methodology and Architecture
5개 DB 검색; QUADAS-2 비뚤림 평가; 51개 연구; SR + MA; 영구치·유치·교합·인접면 층별 분석

## 4. Key Results and Benchmarks
영구치 교합면 (조직학 기준):
- DIAGNOdent Pen (DD Pen): Se 0.81–0.89, Sp 0.71–0.80 ← 최고 Se
- ICDAS-II: Se 0.62–1.0, Sp 0.50–0.84
- DIAGNOdent (DD): Se 0.48–1.0, Sp 0.54–1.0
- Bitewing X-ray: Se 0–0.29, Sp 높음 ← 조기 병소 Se 매우 낮음

## 5. Limitations and Future Work
연구 이질성 높음; QUADAS-2 비뚤림 위험 중간; 탐지 임계값 표준화 필요

## 7. Glossary
- Se (Sensitivity, 민감도): 병소 있을 때 양성 판정 비율
- Sp (Specificity, 특이도): 병소 없을 때 음성 판정 비율
- DIAGNOdent: KaVo 레이저 형광 우식 탐지기
- ICDAS (International Caries Detection and Assessment System, 국제우식탐지평가시스템)
