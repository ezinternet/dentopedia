## Why Ingested

ISQ/RFA 2026 surveillance에서 추출한 신규 RCT. 기존 isq 카테고리의 macro-design→1차 안정성 근거([[implants/isq/stoilov-2023-macrodesign-length-diameter-bone-quality-isq]], [[implants/isq/nandini-2022-cylindrical-vs-tapered-implant-isq]])는 대부분 healed ridge·in vitro 기반인데, 본 RCT는 thread depth 효과를 **즉시식립+fully guided** 라는 복잡 시나리오에서 직접 검증해 그 외삽 가능성을 한정한다. 또 [[implants/isq/konuklu-2026-five-osteotomy-protocols-isq-rct]](osteotomy→ISQ)와 짝을 이뤄 "무엇이 immediate ISQ를 정하는가"의 implant-design 축을 보강. thread depth가 IT는 올리지만 longitudinal ISQ·MBL은 동등하고 오히려 angular deviation을 키운다는 결과는 digital-workflow 정확도 논의([[digital-workflow]])와도 연결.

## One-line Summary
Single-blinded RCT (n=54, 3 arms) in fully guided immediate placement: deeper thread depth raised insertion torque (more sites ≥40 Ncm) but did NOT improve longitudinal ISQ/ISV or 12-mo MBL, and significantly increased angular deviation (MST vs MDT p=0.029), worst in mandibular molars.

## 한줄요약
완전가이드 즉시식립 RCT(54개, 3군): thread depth를 깊게 하면 식립토크는 올라가도(≥40 Ncm 비율↑) 시간에 따른 ISQ·ISV와 12개월 변연골은 동등했고, thread depth만 다른 두 군 비교에서 각도편차는 오히려 유의하게 커졌다(p=0.029, 특히 하악 대구치).

## 1. Document Information
- RCT, single-blinded, 3-arm parallel; Yonsei University Dental Hospital (IRB 2-2020-0101). Aug 2021–May 2025. CONSORT 2025. Trial reg KCT0009402 (retrospective).
- Arms (18 each): BLT = reference tapered implant (different system → system-level comparator); MST = standard thread; MDT = deep thread. MST vs MDT differ ONLY in thread depth (the clean contrast). All self-tapping, tapered, SLA.
- Funding: MegaGen Implant Co. + NRF Korea. Conflict: none declared (MegaGen is implant manufacturer — note funding source).

## 2. Key Contributions
- First clinical RCT to test thread depth on BOTH primary stability and positional accuracy specifically under immediate + fully guided placement (prior evidence = healed ridge / FEA / synthetic bone).
- Separates a within-design thread-depth contrast (MST vs MDT) from system-level noise (BLT), so the thread-depth signal is interpretable.

## 3. Methodology and Architecture
- n=54 immediate implants after extraction, sockets allowing ≥20 Ncm IT. Fully guided flapless (CBCT+IOS merge, R2GATE/Megagen, non-metal tripod guide). 3 freehand conversions excluded from accuracy, kept in ITT stability.
- Socket management predefined: spontaneous (gap ≤2 mm) 25.9% / gap-defect graft (gap >2 mm) 48.1% / GBR (dehiscence ≥3 mm, BioGide) 25.9%; balanced across arms.
- Stability: insertion torque (intraop), ISQ (Osstell), ISV (Any-Check damping) at placement + 1/3/6/12 mo. Linear mixed-effects (no group×time interaction). Accuracy: plan vs postop IOS superimposition (Geomagic Control X) → platform/apex/vertical/angular. MBL: 12-mo PA radiograph, absolute level.
- Sample size: effect size 0.65, α 0.05, power 95% → 14/grp, enrolled 18.

## 4. Key Results and Benchmarks
- Insertion torque: BLT 35.28 / MST 40.00 / MDT 43.89 Ncm (p=0.062); ≥40 Ncm in 14/18 MDT vs 12/18 MST vs 8/18 BLT.
- ISQ/ISV: no group difference at any timepoint (all p>0.30); steady rise over time (largest 1→3 mo, 3→12 mo); 12-mo ISQ ~81–85 all arms.
- Positional accuracy: angular MDT 5.57° vs MST 3.58° (overall p=0.062; **matched MST vs MDT subgroup p=0.029**); horizontal/vertical/platform/apex no sig diff. 4 largest angular deviations (9.2–10.4°) all MDT mandibular molars.
- MBL 12-mo: BLT −0.40 / MST −0.68 / MDT −0.53 mm (p=0.65), all coronal to platform.
- 3 early failures (MST 1, MDT 2) — attributed to local socket anatomy, not thread design.

## 5. Limitations and Future Work
- Extraction-socket heterogeneity (damage, residual volume, defect morphology) confounds isolation of thread-depth effect.
- BLT comparisons reflect system differences, not thread depth — rely on MST vs MDT.
- Industry funding (MegaGen). Some late ISQ missing (non-removable restorations). Calls for stratified socket classification / controlled defect models.

## 7. Glossary
- ISQ: implant stability quotient (Osstell RFA). ISV: implant stability value (damping/percussion, Any-Check). IT: insertion torque. MBL: marginal bone level. MST/MDT: standard/deep-thread (matched macro-design); BLT: reference tapered system.
