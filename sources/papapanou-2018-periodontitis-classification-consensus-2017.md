## 2. Key Contributions
- Staging(I–IV): 중증도(CAL, RBL, tooth loss) + 복잡도(furcation, mobility)
- Grading(A–C): 진행 속도, 위험인자(당뇨, 흡연), 치료 반응성
- Stage III/IV 구분 기준: 치아 상실 기여 여부

## 3. Methodology and Architecture
체계적 문헌 검색 + 국제 전문가 합의(Delphi).

## 4. Key Results and Benchmarks
1999 분류 대체; 치주염을 병기-등급 체계로 재분류. Stage IV: 저작기능 붕괴(mastication collapse) 포함. 공격성 치주염 범주 폐지.

## 5. Limitations and Future Work
새 분류의 임상 적용 장기 검증 필요.
