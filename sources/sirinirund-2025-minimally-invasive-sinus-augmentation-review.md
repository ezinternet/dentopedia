SR(36개 연구, 2732 상악동): 최소 침습 VSA(27개)·LSA(8개) 기법 분류 — 잔존 골높이 ≥5mm: VSA 적용, <5mm: LSA 적용; 최소 침습 기법 모두 충분한 골증량 및 임플란트 성공 달성.

## 3. 최소 침습 기법 분류

### VSA (Vertical Sinus Augmentation, 잔존 골 ≥5mm)
| 기법 | 특징 |
|---|---|
| 스토퍼 회전기 | 깊이 조절 가능, 맹목적 조작 개선 |
| 풍선법 | 수압으로 막 거상, 시각화 제한 |
| 수압법 | 생리식염수 압력으로 막 박리 |
| 디지털 계획 | 가이드 기반 정밀 식립 |
| 내시경 | 직접 시각화, 막 확인 가능 |
| 수술 현미경/루페 | 확대 시야, 정밀도↑ |

### LSA (Lateral Sinus Augmentation, 잔존 골 <5mm)
- 최소화된 창 크기 + 플랩 최소화
- 디지털 계획·가이드 수술 병용

## 4. Key Results
- 모든 최소 침습 기법: **충분한 골증량 및 임플란트 성공** 달성
- 합병증 및 이환율 감소 가능성 — 단 직접 비교 연구 부족
- 천공 발생률: 대부분 연구에서 미보고 → 근거 한계

## 5. Clinical Implications
- 잔존 골 ≥5mm: VSA 우선 — 최소 침습, 동시 식립 가능
- 잔존 골 <5mm: LSA 불가피, 최소 창 + 디지털 계획으로 최소 침습화
- 내시경/현미경: 막 상태 실시간 확인 → 천공 조기 발견 가능

## 7. Glossary
- **VSA**: 경치조(수직) 상악동 거상술; 치조정 경유
- **LSA**: 측창 상악동 거상술; 측벽 경유
- **Minimally invasive SA**: 최소 침습 상악동 거상 — 작은 창, 최소 플랩, 디지털 지원
