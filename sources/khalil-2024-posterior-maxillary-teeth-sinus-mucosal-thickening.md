## Why Ingested

구체적 치아 위치, 치근-상악동 인접도, 치주골 소실 정도를 변수로 상악동 점막비후 위험을 로지스틱 회귀로 정량화한 연구. 제1대구치가 가장 높은 위험(AOR=3.97), 치근단 병변이 압도적 위험(AOR=32.7)임을 제시. [[sinus-lift/lateral/maska-2017-implant-grafting-success-mucosal-thickening-sinus]]의 치주질환 연관성을 다변량 분석으로 더욱 세분화.

## One-line Summary

Retrospective CBCT study (n=124 patients, 992 teeth) quantified risk factors for maxillary sinus mucosal thickening: first molars (AOR=3.97), periapical lesions (AOR=32.7), and severe periodontal bone loss (AOR=2.75) were strongest predictors.

## 한줄요약

CBCT 후향적 연구(n=124명, 992치): 제1대구치(AOR=3.97), 치근단 병변(AOR=32.7), 중증 치주골 소실(AOR=2.75)이 상악동 점막비후의 강력한 위험인자.

## 1. Document Information

- **Type**: Retrospective cross-sectional study
- **Journal**: Cureus (2024, published 2025)
- **DOI**: [10.7759/cureus.76578](https://doi.org/10.7759/cureus.76578)
- **Sample**: 124 patients, 248 maxillary sinuses, 992 teeth; Pharos University, Egypt; age 25-55y

## 2. Key Contributions

- Quantified AOR for specific tooth positions (1st premolar vs 1st/2nd molar vs 2nd premolar)
- Applied Sharan's root tip proximity (RPMT) classification (Types 0-4) with logistic regression
- Demonstrated that periapical lesions have by far the highest AOR for mucosal thickening (32.7)
- Recommends pre-treatment CBCT screening for high-risk cases (second molars, periodontal involvement)

## 3. Methodology and Architecture

- Retrospective cross-sectional CBCT, Pharos University (Feb 2023–Feb 2024)
- Morita 3D CBCT, voxel 0.2mm
- Mucosal thickness graded: Grade 1=0-2mm, Grade 2=2-10mm, Grade 3=>10mm
- Root proximity (RPMT) classified 0-4 (Sharan et al.):
  - Type 0: no contact with sinus cortical borders
  - Type 1: root apex in contact with cortical borders, inferiorly curving floor
  - Type 2: root apex laterally adjacent to sinus cavity, apex outside sinus
  - Type 3: root apex protruding into sinus, inferiorly curving floor
  - Type 4: sinus floor enclosing root apex
- Periodontal bone loss: Type 1 <25%, Type 2 25-50%, Type 3 >50% (Engebretson classification)
- Multivariable logistic regression; inter-examiner κ=0.97

## 4. Key Results and Benchmarks

**MT prevalence**: 72.6% (90/124 patients); 33.9% bilateral

**MT grade distribution** (of 992 teeth):
| Grade | Range | Prevalence |
|---|---|---|
| Grade 1 (normal) | 0-2mm | 72.5% |
| Grade 2 (moderate) | 2-10mm | 19.2% |
| Grade 3 (severe) | >10mm | 8.4% |

**Independent risk factors (AOR, 95% CI):**
| Factor | AOR | 95% CI |
|---|---|---|
| 1st molar vs 1st premolar | 3.97 | 1.82-3.10 |
| 2nd molar vs 1st premolar | 2.30 | 1.29-2.79 |
| 2nd premolar vs 1st premolar | 1.57 | 1.27-2.84 |
| RPMT Type 2 vs Type 0 | 2.27 | 1.38-3.34 |
| RPMT Type 1 vs Type 0 | 2.24 | 1.54-3.26 |
| Severe periodontitis vs none | 2.75 | 1.29-5.86 |
| Mild-moderate periodontitis vs none | 1.68 | 1.19-4.93 |
| Periapical lesion present | 32.7 | 4.53-83.61 |
| No periapical lesion vs extracted | 19.8 | 8.18-74.23 |
| Gender | NS | — |

- 44% of teeth had root proximity Types 1-4 (contact or near contact with sinus floor)
- 80.2% of teeth had mild-moderate periodontal bone loss; 82.1% had no periapical lesion

## 5. Limitations and Future Work

- Age range 25-55 only — older populations (higher periodontitis burden) excluded
- Single center, Egyptian population
- Retrospective — causation cannot be established
- CBCT at single time point — no longitudinal change tracking

## 7. Glossary

- **RPMT (Root tip Proximity to sinus floor - Maxillary Teeth)**: Sharan classification Types 0-4
- **Odontogenic maxillary sinusitis (OMS)**: sinus inflammation triggered primarily by dental sources
- **Periapical lesion → MT pathway**: bacterial toxins/cytokines spread via porous bone, blood, and lymphatics to sinus mucosa
