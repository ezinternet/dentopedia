## One-line Summary
21년간 단일 술자의 4,680개 임플란트 코호트에서 당뇨(RR 2.75), 두경부 방사선(RR 2.73), PMHRT(RR 2.55), 흡연(RR 1.56), 고령(RR 2.24)이 유의한 실패 위험인자로 확인; 고혈압·심질환·스테로이드는 유의하지 않음.

## 1. Document Information
- **Journal**: International Journal of Oral & Maxillofacial Implants 2005;20:569-577
- **Type**: Retrospective cohort study (단일 술자, UCLA)
- **기간**: 1982년 12월 ~ 2003년 7월 (21년)
- **규모**: 4,680개 임플란트, 1,140명 환자

## 2. Key Contributions
- 대규모 단일 술자 코호트로 선택 편향 최소화
- 임플란트 실패 위험인자의 정량적 RR 제시
- 절대 금기사항 없음 확인 — 상대적 위험도 기반 치료계획 수립
- 흡연 실패: 주로 1년 내 / 당뇨 실패: 장기간 지속 / 방사선 실패: 2년 내 집중

## 3. Methodology
- 단일 외과의, 표준화된 술식, UCLA
- 실패 정의: 임플란트 제거로 이어진 모든 상태 (동요, 통증, 감염, 골소실 >50%)
- 분석: 단변량 + 다변량 logistic regression; Relative Risk 산출

## 4. Key Results

### 부위별 실패율
| 위치 | 실패율 |
|---|---|
| 상악 전체 | 8.16% |
| 하악 전체 | 4.93% |
| 하악 전치부 | **2.89%** (최저) |
| 상악 구치부 (좌우) | 8.88–9.66% (최고) |

### 주요 위험인자 RR (단변량)
| 위험인자 | RR | 유의성 |
|---|---|---|
| 연령 60–79세 | **2.24** | p < 0.05 |
| 당뇨 | **2.75** | p < 0.05 |
| 두경부 방사선 | **2.73** | p < 0.05 |
| PMHRT (폐경 후 HRT) | **2.55** | p < 0.05 |
| 흡연 | **1.56** | p < 0.05 |
| 상악 vs 하악 | **1.79** | p = 0.001 |
| 고혈압 | 0.95 | NS |
| 심질환 | 1.02 | NS |
| 스테로이드 | 0.73 | NS |
| 항암화학요법 | 0.63 | NS |

### 다변량 분석 (독립적 예측인자)
- 당뇨: RR 1.94 (p = 0.003)
- 흡연: RR 1.39 (p = 0.03)
- 두경부 방사선: RR 1.87 (p = 0.05)
- 상악 위치: RR 1.79 (p = 0.001)

### 실패 시점 패턴 (Life table)
- **흡연**: 실패의 대부분이 **1년 내** 집중
- **당뇨**: 초기부터 **장기간 지속** (10년 이상)
- **방사선**: **2년 내** 집중, 이후 감소

## 5. Limitations
- 후향적 코호트 (선택 편향 가능)
- 머신 표면 임플란트 비율 높음 (현재 rough surface 결과와 차이 가능)
- 흡연 중단 프로그램 효과 미평가
- PMHRT: 이 당시 호르몬 요법 처방 패턴이 현재와 다름

## 7. Glossary
- **PMHRT**: Postmenopausal Hormone Replacement Therapy (폐경 후 호르몬 대체요법)
- **RR (Relative Risk)**: 노출군에서의 실패 확률 / 비노출군 실패 확률
- **Life table analysis**: 시간에 따른 생존율 추적 분석
- **단일 술자 코호트**: 술자 변수 통제 → 환자·전신 요인 효과 분리 가능
