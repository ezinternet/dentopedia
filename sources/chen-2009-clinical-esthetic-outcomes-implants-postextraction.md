## One-line Summary
ITI 4차 컨센서스 논문(91개 연구): 즉시식립(Type 1)·조기식립(Type 2/3) 모두 생존율 >95%; 단, Type 1에서 순측 점막 퇴축 빈번 — 얇은 생물형·순측 임플란트 위치·순측골벽 손상이 주요 위험인자.

## 1. Document Information
- **Journal**: Int J Oral Maxillofac Implants 2009;24(Suppl):186-217
- **Institution**: University of Melbourne (Chen) / University of Bern (Buser)
- **Type**: 체계적 문헌고찰 (ITI 4차 컨센서스 컨퍼런스, Stuttgart 2008)
- **Sample**: 91개 연구 (1,107 초록, 170 전문 검토 → 포함 기준 충족)

## 2. Key Contributions
- **ITI Type 1-4 분류 체계** 정립 및 검증 (rigid time frames 대신 치유 단계 기준)
- Type 1(즉시)과 Type 2(조기) 간 유사한 생존율 확인
- **Type 1 위험인자** 최초 체계화: 얇은 생물형, 순측 임플란트 오식립, 손상된 순측골벽
- 발치 후 임플란트 식립 타이밍 결정의 근거 프레임워크 제공

## 3. ITI 분류 체계 (Chen & Buser 2008)

| 분류 | 기술적 명칭 | 임상적 목표 치유 상태 |
|---|---|---|
| **Type 1** | Immediate placement | 발치 당일; 골·연조직 치유 없음 |
| **Type 2** | Early placement — soft tissue healing | 연조직 치유 완료, 유의한 골충전 없음 (4-8주) |
| **Type 3** | Early placement — partial bone healing | 연조직+부분 골치유 (12-16주) |
| **Type 4** | Late placement | 완전 치유된 발치와 (>6개월) |

## 4. Key Results

### 재생 결과 (Regenerative Outcomes)
- 대부분 연구에서 골증대술(DBBM ± 차폐막) 후 발치와 결손부 현저한 골충전 확인
- 즉시식립(Type 1) + 조기식립(Type 2)이 지연식립(Type 4)보다 골충전 효과 우수
- GBR 기법 간 우열 불분명 (이종골, e-PTFE, 콜라겐막 등 다양)

### 생존율 (Survival)
- 대부분 연구에서 **생존율 >95%**
- Type 1 vs Type 2: 생존율 유사
- Type 4(지연식립): 생존율 비열등, 단 최적 골재생 효과 낮음

### 심미 결과 (Esthetic Outcomes)
- **Type 1(즉시식립)**: 순측 점막 퇴축 빈번 → 평균 0.5-1mm 퇴축
- **Type 2/3(조기식립)**: Type 1 대비 퇴축 빈도 낮음
- **Type 1 위험 3대 인자**:
  1. 얇은 연조직 생물형(thin tissue biotype)
  2. 순측 임플란트 오식립(facial malposition)
  3. 얇거나 손상된 순측골벽(thin/damaged facial bone wall)

## 5. Clinical Implications
- Type 1 식립은 엄격한 환자 선택 필수 (후벽 생물형, 온전한 순측골 선호)
- Type 2 식립: 연조직 치유로 GBR 공간 폐쇄 용이 + 심미 위험 감소
- 순측골벽 손상 시 Type 2/3 고려
- 모든 타입에서 골증대술 병행 권고

## 7. Glossary
- **ITI Type 1-4**: 발치 후 임플란트 식립 타이밍 분류 (즉시~지연)
- **Thin tissue biotype**: 얇은 연조직 생물형; Type 1 후 퇴축 고위험
- **Facial malposition**: 임플란트가 순측으로 치우쳐 식립된 상태 → 퇴축 유발
- **DBBM**: Deproteinized bovine bone mineral (이종골); 발치와 충전재로 가장 많이 사용
