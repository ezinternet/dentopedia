## One-line Summary
Case series (서울대 치병원, J Craniofac Surg 2024): sinus pathology 3 type을 type 1 (PsC), type 2 (MRC <20mm), type 3 (MRC >20mm)로 분류하고 각각 spontaneous drainage·aspiration·enucleation을 시행. **Subtype별 결정 트리** 임상 예시.

## 1. Document Information
서울대 치과병원 OMFS 그룹 (이원욱·한정준·권익재). J Craniofac Surg, IF 1.0 정도. 명확한 의사결정 트리 framework이 본 case series의 가치.

## 2. Key Contributions
- Sinus pathology subtype별 결정 트리 명시
  - Type 1 (PsC): spontaneous drainage → sinus augmentation
  - Type 2 (MRC <20mm): aspiration → sinus augmentation
  - Type 3 (MRC >20mm): enucleation → 회복 후 sinus augmentation
- 3 case 동시 보고 — subtype별 술식 직접 비교 framework

## 3. Methodology
- Case series, 3 patient
- Pathology subtype 분류: PsC (type 1), MRC <20mm (type 2), MRC >20mm (type 3)
- 각 case에 적합한 술식:
  - Type 1: 자연 배출 후 단순 augmentation
  - Type 2: 흡인 후 augmentation
  - Type 3: enucleation → 완전 회복 → augmentation
- 추적: implant 안정·sinus health

## 4. Key Results
- 3 case 모두 성공적 implant 식립·정착
- Subtype별 술식 차등 적용 — 임상적 framework로 작동

## 5. Limitations
- n=3 — 통계 결론 불가
- 단일 그룹·실험 디자인 부재
- "20mm" 기준선의 근거는 본 case에서 가설적 — RCT validation 없음
