## One-line Summary
전방부 복합레진 수복의 수리(repair) vs 교체(replacement) 후향적 15년 연구(634 수복): 수리가 생존율을 높이는 적합한 대안.

## 1. Document Information
- **Title**: Is composite repair suitable for anterior restorations? A long-term practice-based clinical study
- **Authors**: van de Sande FH, Moraes RR, Elias RV, Montagner AF, Rodolpho PA, Demarco FF, Cenci MS
- **Year**: 2019 (online: October 2018)
- **Journal**: Clinical Oral Investigations 23:2795–2803

- **Study Type**: retrospective
- **Category**: resin

## 2. Key Contributions
- 수리를 실패로 보지 않을 때: 15년 Class III/IV 생존율 69%, AFR 2.4%
- 수리를 실패로 볼 때: 15년 생존율 64%, AFR 2.9%
- 직접 베니어: 5년 85%, 10년 74%로 하락 (수리=실패 시)
- 상악 수복이 하악보다 실패 위험 높음 (p<0.01)
- 중절치가 견치보다 실패 위험 높음 (p<0.01)

## 3. Methodology and Architecture
- 개인 치과 진료 데이터(1994–2009), 144 환자 634 수복
- 수리를 실패/비실패로 나누어 2가지 분석
- Kaplan-Meier, Cox regression, log-rank test

## 4. Key Results and Benchmarks
- Class III/IV 15년 생존율: 69% (repair≠failure) vs 64% (repair=failure)
- AFR: 2.4% vs 2.9%
- 직접 베니어 5년 → 10년: 85%→74% (수리≠실패) or 52%→38% (수리=실패)
- 수리: 치질 보존, 의자 시간 단축, 환자 불안 감소

## 5. Limitations and Future Work
- 단일 개인 진료실 데이터 → 일반화 주의
- 후향적 관찰 연구 — RCT 아님
- 베니어 수복 수 제한적
