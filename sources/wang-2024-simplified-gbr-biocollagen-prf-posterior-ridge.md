## One-line Summary

Retrospective study (n=44, 2 groups) comparing simplified GBR (Bio-collagen + PRF membrane) vs classical GBR (Bio-Oss + Bio-Gide) for posterior Seibert Class I ridge defects: HW gain 1.50 mm vs 1.83 mm at 6 months; comparable contour morphology.

## 한줄요약

후향적 연구(n=44)에서 Bio-collagen+PRF막(단순화 GBR) vs Bio-Oss+Bio-Gide(고전적 GBR)로 후방 Seibert Class I 치조제 결손 치료: 수평폭경 증가 1.50 vs 1.83mm, 윤곽 형태 유사.

## 1. Document Information

- **Type**: Retrospective comparative study
- **Journal**: Tissue Engineering and Regenerative Medicine 2024;21(6):959-967
- **Note**: Abstract-only — full text not retrieved from PMC

## 2. Key Contributions

1. **Simplified GBR feasibility**: Bio-collagen (BC) + PRF membrane achieves clinically significant horizontal ridge width gain in posterior Seibert Class I defects — avoids expensive Bio-Oss/Bio-Gide combination for minor defects
2. **Contour morphology classification**: Type A (good contour) predominated in both groups (14/22 simplified vs 11/22 classical); Type C (poor) minimal in both
3. **HW gain**: 1.50 mm (simplified) vs 1.83 mm (classical) — classical GBR numerically superior but clinical significance of 0.33 mm difference requires contextualization
4. Supports PRF as barrier membrane substitute in low-demand posterior ridge augmentation scenarios

## 3. Methodology

- **Design**: Retrospective, n=44 (22 per group)
- **Group 1 (Simplified GBR)**: Bio-collagen graft + PRF membrane cover
- **Group 2 (Classical GBR)**: Bio-Oss xenograft + Bio-Gide resorbable collagen membrane
- **Outcome**: CBCT at 6 months; horizontal width (HW) measurement; buccal ridge contour Type A/B/C classification
- **Population**: Posterior Seibert Class I ridge defects

## 4. Key Results and Benchmarks

- **HW increase**: Simplified GBR 1.50 mm vs Classical GBR 1.83 mm (difference 0.33 mm)
- **Contour Type A**: 14/22 (64%) simplified vs 11/22 (50%) classical
- **Contour Type C (poor)**: 1/22 (5%) simplified vs 3/22 (14%) classical — simplified trended better
- Both approaches achieved statistically significant bone augmentation

## 5. Limitations and Future Work

- Abstract-only retrieval; statistical significance of HW difference not confirmed from abstract
- Retrospective design; potential selection bias
- Small n per group (22); posterior region only; Seibert Class I only
- No long-term implant placement outcomes reported
- Full contour classification schema requires full-text verification

## 7. Glossary

- **Seibert Class I**: Horizontal (buccolingual) ridge deficiency with normal height — the simplest ridge defect category
- **Bio-collagen (BC)**: Collagen-based bone substitute; distinct from DBBM (Bio-Oss)
- **PRF membrane**: Platelet-rich fibrin membrane — autologous, prepared chairside; acts as barrier + growth factor depot
- **Contour Type A/B/C**: Classification of post-GBR buccal ridge morphology developed in this study
