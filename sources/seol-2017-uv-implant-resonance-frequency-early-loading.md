## One-line Summary
밀폐형 UV 처리 임플란트 증례보고: UV 광기능화로 초친수성·양전하 표면 구현 → stability dip 없이 ISQ 조기 상승 → 식립 1주~4주 내 조기부하 성공.

## 1. Document Information
- **Journal**: Implantology 2017;21(2):110-127 (대한구강악안면임플란트학회지)
- **Institution**: 중앙대병원 치과보철과, 프라임치과, 굿윌치과병원, 에스플란트치과병원, 미유치과
- **Type**: 증례보고 (Case Report)
- **Sample**: 복수 증례 — 발치와 즉시식립, 상악동 동시거상, 하악 조기부하 포함

## 2. Key Contributions
- UV 광기능화(photofunctionalization)의 임상 적용 증례 보고
- **Stability dip 비출현**: UV 임플란트에서 ISQ가 식립 직후부터 지속 상승
- **조기부하 성공**: 식립 후 1주 이내 부하에서도 ISQ 안정 유지
- 기존 동물실험 결과(BIC 100%, 골유착 강도 3배)의 임상 확인

## 3. UV 광기능화 원리
| 특성 | 일반 Ti | UV 처리 Ti |
|---|---|---|
| 친수성(접촉각) | ~70° | **≈0° (초친수성)** |
| 표면 탄소 조성 | 60-75% | **<20%** |
| 표면 전하 | 음전하 | **양전하** |
| BIC (동물실험) | 55% | **≈100%** |

→ 세포 부착·증식·골형성 현저 촉진; 골유착 강도 3배 증가 (동물실험 기준)

## 4. Key Results (대표 증례)

### Case #1: 발치와 즉시식립 + 조기부하
| 시점 | #36 ISQ | #37 ISQ |
|---|---|---|
| 식립일 | 69 | 74 |
| 1주 | 73 | 75 |
| **2주** | **81** | **81** |
| 3주 | 80 | 81 |
| 4주 (조기부하) | 80 | 80 |

- Stability dip 없음; 2주차에 급격 상승 후 안정 유지
- 4주 후 조기부하, 임상·방사선학적 이상 없음

## 5. Clinical Implications
- UV 처리 → 골유착 신속화 → 조기부하 타이밍 앞당길 수 있음
- ISQ 추적으로 조기부하 시점 결정 (ISQ ≥ 70 이상에서 안전)
- 연질골·발치와 즉시식립 등 고위험 상황에서 UV 처리가 안전 마진 제공
- Stability dip 없는 환자 → 즉시/조기 부하 프로토콜 적용 용이

## 7. Glossary
- **UV photofunctionalization (UV 광기능화)**: 자외선 조사로 Ti 표면 화학성질 변환 → 초친수성·양전하
- **Sealed UV system**: 포장 내에서 UV 조사 완료 → 표면 오염 없이 임상 적용
- **Stability dip**: 식립 후 1차→2차 안정성 전환기의 일시적 ISQ 저하
- **BIC (Bone-to-Implant Contact)**: 골-임플란트 접촉률; 골유착 지표
