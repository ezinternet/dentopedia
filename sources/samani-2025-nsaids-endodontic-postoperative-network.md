## One-line Summary
Network MA (IJPPP 2025, 16 RCT, n=2021): NSAIDs·코르티코스테로이드가 비외과적 근관치료 후 8~72h 전 시점에서 통증 유의 감소; 높은 이질성으로 최적 약제 결론 유보.

## 1. Document Information
Int J Physiol Pathophysiol Pharmacol 2025;17(4):116-130. Scopus·PubMed·Google Scholar 검색(~2024.02). 16 RCT, 2,021명 포함. 다기관 공동 저술(이란·사이프러스·중국).

## 2. Key Contributions
- NSAIDs 및 코르티코스테로이드가 8h, 12h, 24h, 48h, 72h 전 시점에서 통증 유의 감소
- 8h 시점 풀링 효과: -3.10 (I²=100%)
- 72h 시점 효과: -0.64 (I²=73.1%) — 이질성 감소
- Funnel plot 대칭 → 출판 비뚤림 낮음
- 네트워크 메타분석으로 간접 약제 비교 시도

## 3. Methodology
Network meta-analysis. Scopus·PubMed·Google Scholar 체계적 검색. 16 RCT 포함. 무작위효과 모델, I² 이질성 평가, funnel plot, 민감도 분석(72h에서 1편 제외).

## 4. Key Results
| 시점 | 풀링 효과 | I² |
|---|---|---|
| 8h | -3.10 | 100% |
| 12h | -1.69 | 99.2% |
| 24h | -1.48 | 99.9% |
| 48h | -1.42 | 99.4% |
| 72h | -0.64 | 73.1% |

효과크기 시간에 따라 감소 → 초기 통증 억제 효과가 가장 큼.

## 5. Limitations
- I² 최대 100%: 극도로 높은 이질성 → 풀링 해석 주의
- 약제 간 직접 비교 RCT 제한적
- 추가 고질의 RCT 필요

## 6. Related Work
Smith 2017 (ibuprofen SR+MA), Di Spirito 2022 (overview), Pak 2011 (통증 유병률).

## 7. Glossary
Network MA: 직접+간접 비교를 통합한 메타분석; PEP: Postoperative Endodontic Pain; I²: 이질성 지수.
