## One-line Summary
In-vitro (J Adhes Dent 2025): 모노리식 지르코니아 타액 오염 후 세정 6종 비교 — 샌드블라스팅+Ivoclean(9.3 MPa) 최우수; 이소프로판올 최열등; 알코올·물·퍼미스 무효.

## 1. Document Information
J Adhes Dent 2025;27:93-102. doi:10.3290/j.jad.c_2011. Selcuk University (Turkey) + Univ Zurich. 모노리식 지르코니아 105개 시편, 열순환 5,000회.

## 2. Key Contributions
- 샌드블라스팅+Ivoclean 병용이 단독보다 유의 우세 확인
- 이소프로필알코올이 대조군(무처리)보다도 접착강도 낮음 — 임상적 경고
- 퍼미스·물분사: 대조군과 유의차 없음
- Ivoclean 메커니즘: 지르코니아산화물 입자가 타액 인산염 흡수 → 표면 탈오염

## 3. Methodology
인공타액 오염 후 6가지 세정: ①물분사 ②이소프로필알코올 ③퍼미스 ④Ivoclean ⑤샌드블라스팅 ⑥샌드블라스팅+Ivoclean. 자가접착 레진시멘트 접착. 열순환 5,000회. 전단접착강도(SBS), SEM/EDS, ANOVA-Duncan.

## 4. Key Results
| 세정법 | SBS (MPa) |
|--------|-----------|
| 샌드블라스팅+Ivoclean | 9.3 |
| 샌드블라스팅 | 8.59 |
| Ivoclean | 7.21 |
| 퍼미스 | 4.82 |
| 물분사 | 4.15 |
| 대조군(무처리) | 3.65 |
| 이소프로필알코올 | 3.04 |

## 5. Limitations
In vitro; 인공타액 (실제 타액과 다름); 단일 자가접착 레진시멘트; 5,000회 열순환 (임상 10년 추정).

## 6. Related Work
Silva 2022 SR+MA (샌드블라스팅 최우수 확인), Kwon 2026 (SARC 종류별 지르코니아 접착).

## 7. Glossary
SBS: 전단접착강도 (Shear Bond Strength); Ivoclean: 지르코니아 전용 세정제; 모노리식 지르코니아: 단일체 지르코니아 (비니어링 없음).
