## One-line Summary
Case report (Cureus 2025, USP Ribeirão Preto + ARDEC): MRC 환자에서 simultaneous drainage + lateral SFE → guided implant placement. 6년 후 clinical·CBCT·histology 모두 안정 확인.

## 1. Document Information
University of São Paulo (Ribeirão Preto) + ARDEC Academy (Italy) 협력. Cureus open access. 단일 case지만 6년 추적·조직학 분석으로 longitudinal value 높음.

## 2. Key Contributions
- 6년 longitudinal data — sinus pseudocyst cluster에서 드문 장기 추적
- Histology 포함: 신생골·잔존 골이식재 형태 분석
- Guided implant placement과 결합

## 3. Methodology
- Single case
- 술식: MRC drainage → lateral SFE → guided implant 식립 (delayed)
- 평가: clinical·CBCT (tomography)·histology
- Follow-up 6년

## 4. Key Results
- 6년 implant 안정·정착
- CBCT: grafted bone 잘 유지
- Histology: vital bone 형성·잔존 골이식재 분포

## 5. Limitations
- n=1, case report
- 단일 술자·institutional protocol
- Cureus = open access but lower-tier journal
