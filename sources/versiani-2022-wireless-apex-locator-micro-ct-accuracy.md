## One-line Summary
Micro-CT로 Wirele-X와 RootZX II 정확도 생체 내(in vivo) 비교: ±0.5mm 기준 정확도 90.9% vs 81.8%; 통계적 유의차 없음(P=.61) — 두 EAL 임상적 동등.

## 1. Document Information
- **Journal**: Journal of Endodontics 2022;48:1152-1160
- **Institution**: 브라질 (Versiani)
- **Type**: 전향적 in vivo 연구 (micro-CT 분석)
- **Sample**: 5명 환자, 발거 예정 11개 치아

## 2. Key Contributions
- Micro-CT를 이용한 EAL 정확도 3D in vivo 평가 최초 시도 중 하나
- Wirele-X(무선 EAL)와 RootZX II 직접 비교
- ±0.5mm 기준 Wirele-X 90.9% vs RootZX II 81.8% — 통계적 차이 없음
- 기존 연구의 한계(직접 육안 측정, 2D)를 micro-CT 3D 분석으로 극복

## 3. Methodology
- **대상**: 발거 예정 생활치 11개 (5명 환자)
- **방법**: EAL로 근관장 측정(0.0 표시) → 스톱 고정 → 발치 → micro-CT 이중 스캔
  - 파일 있는 상태 / 없는 상태 CT 동시 획득 → 정합 → 파일 팁~major foramen 거리 측정
- **정확도 기준**: ±0.5mm 내 major foramen 감지 여부
- **통계**: χ² test (정확도 비교), paired t-test (양 측정값 차이)

## 4. Key Results
| 기기 | ±0.5mm 정확도 |
|---|---|
| **RootZX II** | **81.8%** |
| **Wirele-X** | **90.9%** |
| χ² P값 | 0.6109 (유의하지 않음) |
| 절대거리 P값 | 0.82 (유의하지 않음) |

- 두 기기 모두 major foramen에서 ±0.5mm 내 측정 가능
- 통계적 유의차 없음 → 임상적 동등성

## 5. Limitations
- 소표본 (11개 치아)
- 단일 연구자
- 0.0 표시 기준 (0.5 표시 기준 연구와 비교 어려움)

## 6. Related Work

## 7. Glossary
- **EAL (Electronic Apex Locator)**: 전기 저항 원리로 근단공 위치 측정 기기
- **Major foramen**: 근관이 치근 표면으로 나오는 주공(主孔)
- **Micro-CT**: 고해상도 3D X선 분석 — 비파괴적으로 파일-근단공 관계 정확 측정
- **Wirele-X**: 무선 EAL; RMS 기반 신호 처리로 노이즈 저감
- **RootZX II (J. Morita)**: 대표적 기준 EAL; 다중 주파수 방식
