## One-line Summary
J Periodontal Implant Sci 2017;47(4):194-210: n=7, DBBM + porcine collagen matrix socket preservation, 6-month histology — width resorption 1.21mm, height 0.46mm; 31.97% residual xenograft, 16.02% new bone; uneventful healing.

## 1. Key Results
- 7 sockets, 7 patients; DBBM + non-crosslinked porcine collagen matrix (CM)
- Soft tissue healing: uneventful in all sockets
- Horizontal resorption: 1.21mm (p=0.005)
- Vertical resorption: 0.46mm (p=0.004)
- Histomorphometry (6 months): residual xenograft 31.97±3.52%; new bone 16.02±7.06%; connective tissue 50.67±8.42%
- CM: physiological substitution to vascularised collagen-rich connective tissue
- University of Milan (IRCCS Cà Granda)
